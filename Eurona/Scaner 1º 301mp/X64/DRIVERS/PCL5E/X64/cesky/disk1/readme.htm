[1] How to install PCL printer driver

(1) Installing PCL printer driver at the first time


* Make sure to close all applications before installing.

- 1 Start Windows and log on as Administrators or Power Users.
When using a driver downloaded from the Internet:
After downloading the driver from the Internet, extract the file(s) to a folder of your choice.
From the [Start] menu, point to [Settings] or [Control Panel], then click [Printers] or [Printers and Faxes] or [View Devices and Printers]. In the Printer folder double click [Add Printer] (it may be in [Printer Tasks]), and click "Next >".

Å@
- 2 Follow the Wizard guidance and select a printer connection method (Local or Network Printer), then click "Next >" to continue.
- 3 The following procedure below would be an explanation when selecting [Local Printer] in the step above.
A) To install the downloaded driver, click [Have Disk] then [Browse] to locate the folder where you had extracted the downloaded driver, from this folder select Disk 1, then click "OK".
B) Insert the CD-ROM into the CD ROM Drive, [Have Disk] button in "Manufacturers:" / "Printers:" window, then enter the path (e.g. D:\DRIVERS\PCL{5,6}\XP_VISTA\(language)\DISK1\) of the designated Drive then click "OK". (You can also navigate to the Drive by clicking the [Browse] button).
- 4 Select the printer port to use.
After selecting, click "Next >".
- 5 Select the printer to use.
After selecting, click "Next >".
- 6 Specify printer name. And if you want to use installed printer as the default printer, check "Yes".
After specifying, click "Next >".

- 7 If you want to share installed printer with other network users, indicate and set sharing.
After above, click "Next >".
- 8 Print test page to confirm that the printer is set up properly. Check on "Yes(recommended)" , then click "Finish", and finally , the installation is completed.
After completing above operation, you can use PCL printer driver for Windows XP/Windows Server2003/Windows Vista/Windows Server2008/Windows 7.


(2) Upgrading the PCL printer driver

Please follow these installation steps to update the printer driver from old version to new version.

* Make sure to close all applications before upgrading.

- 1 Open the "Printers" folder (or "Printers and Faxes", "View Devices and Printers") from the "Start" menu or "Control Panel".

- 2 Open the Properties of the printer to be upgraded.
- 3 Select "Advanced" tab > "New Driver" button. Click "Next >".
- 4 Select "Have Disk..." then browse to the folder containing the extracted printer driver INF file. Click "Open".
- 5 Click "OK" and select the printer model. Click "Next >".
- 6 Once the installation is complete, click "Finish".
- 7 Click "OK" to close the printer properties dialog box.
After completing above operation, you can use PCL printer driver of new version for Windows XP/Windows Server2003/Windows Vista/Windows Server2008/Windows 7.


 (3) Uninstalling the PCL printer driver

 * Make sure to close all applications before uninstalling.
- 1 Open [Printers] or [Printers and Faxes] or [View Devices and Printers] folder in [Settings] from [Start] menu.
- 2 Select the printer you want to uninstall.
- 3 Press [Delete] key.
- 4 Select "Driver" Tab in Server Property from File Menu in Printer Folder.
- 5 Select the driver you want to uninstall.
- 6 Press [Delete] key.

(4) Notes for the installation

* With PCL printer driver, the drivers of different versions can't be used together.
* If "Print Settings" screen doesn't look normal after installation, delete the driver with the procedure of above (3) and reinstall the driver.
* Do not extract CD-ROM before the installation is finished. Or else, system error might occur.
* During the driver installation, Windows may suggest to keep the newer module already installed in Windows. Please select "Yes" to use the newer module.
* The CD-ROM installer causes the Spooler service to temporarily stop. We do not recommend using the CD-ROM installer on a member server of a Microsoft cluster environment because other services that rely on the Spooler might also stop causing a failover to occur. Instead, the Add Printer Wizard should be used. If this is done, the Spooler service will not stop.

(5) Notes for the upgrading

* The settings of overwritten PCL5 drivers will not be inherited.
* An upgrade can only be performed between drivers of the same model and type of PCL (PCL5e, PCL5c).

[2] Limitation/Notes in using PCL5 printer driver

The settings which are introduced below are depending on the printer or the printer driver you use.


 [In General]
* To avoid problems in general, make sure that there is plenty of free disk space.
* With some applications (e.g. CorelDRAW, Visio), complicated images may take a long time to print.
* If the printer driver language differs from that of the Operating System, characters may not be printed properly or the characters on the printer driver may not be displayed properly.


[OS/Windows Related Issues]
* When the spooler is paused, please do not send the print job.
* If a Separator Page is selected, some documents may not print as expected.
* Please do not change the "Driver" setting on the "General" or "Advanced Options" tab.


[With Driver Settings]
* [Copies] in [Setup] Tab:
When "Copies" is selected on the driver's "Setup" tab, the value may not be reflected in the application print dialog box. Be sure to set the number of copies only in the driver, not in the application.
* [Staple] in [Setup] Tab:
Documents may not be stapled at the slant angle, even when you select "Top Left Slant" or "Top Right Slant". In such a case, the document will be stapled parallel to an edge of the paper.

* [Custom Paper Size] in [Paper] Tab:
The measurement unit in the Custom Paper Size dialog is reset to "cm" when the driver dialog is reopened, even if the custom paper size was previously specified in "inch". The paper size specification is retained regardless of the unit.

* [On first page only] in [Watermark] Tab, and [Booklet/Duplex] in [Setup] Tab:
The watermark is not printed on the first page when both "On first page only" and "Booklet" are selected.

* [Print Quality] Tab:
If the "TrueType Mode" is set to "Download as bit image", the watermark may not print correctly.
* [Booklet/Duplex] in [Setup] Tab:
Printing a large document in booklet, duplex or reverse order printing produce a large temporary file. If you encounter an error when printing a large document in booklet, try again after increasing the free area in system installed drive (e.g. C:).

* [Edge to Edge Print] in [Print Quality] Tab, and [Non Reduction Booklet] in [Setup] Tab:
When both "Edge to Edge Print" and "Non Reduction Booklet" funtions are selected, some documents may not print as expected.

* [Covers] in [Paper] Tab:
Use of a different Paper Source for the cover setting might result in a paper mismatch error. This occurs if the cover is of a different paper size than the rest of the document. This only happens if the cover is not blank and if the source tray for the rest of the document is not "Auto Tray Select". As a workaround, change the print settings so that any one of the above conditions are not met. Note: "cover" = Front Cover, Back Cover or Chaptering Pages

* [Print Banner Page] in [Valid Access] Tab:
The first 15 bytes of the hostname can be printed on the banner page.
Banner pages cannot be printed using the source tray that is selected to print "Chapter Page". Please use another source tray to print banner pages.


[Graphics/Images]
* When graphics images or text printed as graphics images are not printed as they appear on your computer screen, change the "Graphics Mode" on the "Print Quality" tab.
* Selecting the Spool Type as EMF may cause some applications not to print well in "Vector" mode. If this happens, return the Spool Type setting to RAW.


[Fonts/Texts]
* With Adobe Type Manager(ATM), some characters may not print correctly. To avoid this problem, print the document with PCL 6 or PostScript driver.
* With some applications, some characters may not print correctly. To solve this problem, change the "TrueType Mode" and/or the "Use Printer Fonts" on the "Print Quality" tab.
* Documents that contain fonts not installed on your computer will not print correctly. Make sure your document only contains fonts installed on your computer.
* The watermark as "As Outline only" may not print if the watermark message contains Chinese characters. To avoid this problem, use the PCL 6 driver.
* Additional lines may appear when the bold HANYI fonts larger than 36 points are printed.
* The HANYI fonts can not be printed in 300 dpi. To avoid this problem, print it in 600 dpi.
* To print watermark text in Chinese, choose a Chinese font.


[Connection/Port related Issues]
* When printing complicated documents to a printer connected with a parallel interface, the printer time-out may cause the document to print incorrectly. If this happens, set the Spool Setting to "Start printing after last page is spooled" and/or extend the IO timeout at the printer operation panel(max. 300 seconds).
* When the printer is connected with a SMB port, "Update Now" in the Accessories tab doesn't work. Please manually configure the accessories.


[CMYK used for Black and White Documents]
* Black and White documents might be printed in full color (CMYK).
To ensure that the job prints using pure Black (K), please change the color mode to "Black and White".
* When selecting "Color" from "Color/ Black and White" setting in the "Setup" Tab, black and white document may be printed as a color job, depending on application or data. If you want to print a document in black and white, select "Black and White" in "Color/ Black and White" setting.


[Help issues]
* Windows Vista does not support the help format used in previous versions of Windows. Please download the legacy help application WinHlp32.exe from Microsoft.
* When resolution setting of Display is smaller than 800x600dpi, "Information Help" window dialog may be oversized on the display. In this case, to close the window, click "X" botton on the right corner of the window dialog.


[Other issues]
* Additional Page Count in Duplex Printing
An extra page is added to the end of odd-paged duplex jobs. The added page might be incremented in the printer's counter. 
* Point&Print drivers do not inherit Watermark settings
Drivers installed on a client using Point&Print do not inherit Watermark settings from the server.
* Reverse Order is not supported with the Duplex, Front Cover, Back Cover, Chapter Pages, Slip Sheet functions.


[Application issues]
* Microsoft
* Microsoft Office XP *
- If you are using Office XP, be sure to apply Service Pack 1 or later to avoid printing problems.
* MS-Project *
- When using with MS-Project, the stripe pattern interval may not print as it appears on your computer screen. If this happens, change the "Graphics Mode".
- To print multiple copies from MS-Project, set the number of copies setting in the application print dialog box to 1, and then specify the number of copies using "Copies" setting on the driver's "Setup" tab.
* PowerPoint *
- PowerPoint may print some graphics objects with the same color although they are in different colors on the computer screen. To solve this problem, change the object's color.
* WordPad
- When you change the printer properties from WordPad, changes will not be reflected in the print job immediately following the change.
* Corel
* CorelDRAW *
- You can not enter the number of copies greater than 9 in the printer driver dialog opened from CorelDRAW 8, due to a problem in CorelDRAW 8. To print more than 9 copies, set the number of copies in the CorelDRAW 8's print dialog.
- The printer driver dialog opened from CorelDRAW 8 may show unreadable characters, due to a problem in CorelDRAW 8.
* Adobe
* Acrobat *
- If printed using Adobe Acrobat/Acrobat Reader, black and white PDF documents compressed by MMR are printed in full color with some image distortion/deterioration. This can be prevented by disabling the application's "Print As Image" function.
- Adobe Reader might add an extra page of padding to the end of odd-paged duplex documents. This will be counted as a BW page.
When application's copies is 1 and PCL5 driver's copies is 2 or more, the extra page is not added.
* Frame Maker *
- With Adobe Frame Maker, please use *.FM document to print when "Duplex/Booklet" or "Staple" function is selected.
* Page Maker *
- PageMaker may not print graphics if a document contains large amount of graphics. To print such documents, change the "Memory Usage" settings to "Font Priority" on the printer operation panel.
- PageMaker shifts the image by about one inch with some paper sizes. To solve this problem, print using a different paper size.
