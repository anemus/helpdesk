Getting Started

Installing the printer driver

Be sure to close all applications before upgrading the driver. About the procedure of installing with [Add a Printer] function that provided by Windows, please check the Help of the operating systems.

  1. Run the installer EXE file. These files and this README file are included in the same directory.
    1. for 32Bit：RP_SETUP32.EXE
    2. for 64Bit：RP_SETUP64.EXE
  2. The "License Agreement" dialog will be displayed. Check them all and if you agree with them, select "I accept the agreement" and click "Next".
  3. The "Method to install printer driver" dialog will be displayed. select "Add a new printer" and click "Next".
  4. The "Select Method to Add Printer" dialog will be displayed.
    1. To show network connected printers, select both of "Connect to a printer port or LAN" and "Search for printers automatically" and click "Next". Go on to the Step5.
      ("Search for printers automatically" might not selectable depending on the printer type.)
    2. If the printer's IP address is given, select both of "Connect to a printer port or LAN" and "Search for printers by specified IP address" and click "Next". Go on to the Step6.
      ("Search for printers by specified IP address" might not selectable depending on the printer type.)
    3. If the port of printer is already set up, or local port is desired, select both of "Connect to a printer port or LAN" and "Change the printer port after completing installation" and click "Next". Go on to the Step7.
      ("Change the printer port after completing installation" might not selectable depending on the printer type.)
    4. USB connetion is desired, select "Connect USB cable" and click "Next". Go on to the Step8. ("Connect USB cable" might not selectable depending on the printer type.)
  5. Select the printer and click "Next". Go on to the Step7.
  6. Input IP address and click "OK". Go on to the Step7.
  7. Select the printer. To change the name of printer icon, input desired name to "Change settings for 'printer name'". To customize more settings, click '+' symbol. If the user code is configurable, printer side's print counter function is available.
  8. Click "Continue". File copying will be started. Excepting USB Setup, Go on to the Step12.
  9. The "Select USB Connected Printer" dialog will be displayed. Select printer and click "Next".
  10. Check cable and power switch and click "Next". File copying will be started and automatically proceed to Step11.
  11. The "Auto-detect USB Port" dialog will be displayed. Connect USB cable and turn power switch ON. Device will be detected and automatically proceed to Step12.
  12. The "Printer Driver Installation Complete" dialog will be displayed. Click "Finish".

Upgrading the printer driver

If the printer driver is already installed, upgrade the printer driver.

  1. Run the installer EXE file. These files and this README file is included in the same directory.
    1. for 32Bit：RP_SETUP32.EXE
    2. for 64Bit：RP_SETUP64.EXE
  2. The "License Agreement" dialog will be displayed. Check them all and if you agree with them, select "I accept the agreement" and click "Next".
  3. The "Method to install printer driver" dialog will be displayed. Select "Upgrade the current printer" and click "Next".
  4. Upgrading targets will be displayed. Confirm and click "Next". File copying will be started.
  5. The "Printer Driver Installation Complete" dialog will be displayed. Click "Finish" and restart your computer.

Other important points

[Installer Related Issues]

  1. After the installations, if the printer setting dialogs seem abnormal, delete the printer driver and install it again.
  2. To be available printer's functions, set up Accessories. Accessories settings are required if the bi-directional communication is not activated. It might not necessary depending on the printer type.
  3. Installing with this installer might replace RICOH printer driver's print processor settings if they are already installed to system and customized. In such case, customize settings again or install with [Add a Printer] function provided by Windows.
  4. If following massage is displayed during install sequence, this installer can't complete installing.
    [There is already a more recent version of this Driver installed on the system.]
    Install with the driver that is described in above message.

[OS/Windows Related Issues]

  1. Log on as Administrators to use this installer program.
  2. When the computer is turned on or logged on, following messages might be displayed. Their descriptions are depends on OS type. If these messages are displayed, click [Cancel].
    1. [Found New Hardware]
    2. [Found New Hardware Wizard]
    3. [Add Hardware Wizard]
  3. During printer driver's installation, confirmation dialog of digital signature might be displayed. But there is no problem for its function, so continue installation.
  4. If the Microsoft Minidriver is installed to OS, when the printer is connected to the computer with parallel port, OS might install its Minidriver automatically.
  5. In the case of WindowsServer2003, if the bi-directional communication is activated, opening printer properties dialog might take long. Applying ServicePack1 or later is recommended.
  6. In the [Select and set Printer] dialog, printer port setting might be empty. If "Continue" is clicked with no change, following message will be displayed.
    [The requested Port does not exist.(44)]
    Select port and continue installation.
  7. Do not use this installer on the environment of Microsoft Cluster Server. During the installation procedure, spooler service will be stopped temporary and this might touch other services. Install with [Add a Printer] function that provided by Windows surely. Depending on the driver's version, this installer might unavailable and following message will be displayed.
    [Not compatible with this version of the OS.(8)]
