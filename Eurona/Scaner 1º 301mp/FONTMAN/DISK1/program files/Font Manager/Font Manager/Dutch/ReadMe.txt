1.0 INLEIDING
Met lettertypen brengt u gevoel en betekenis in teksten en presentaties. Met Font Manager beheert u gemakkelijk lettertypen in MS Windows. In één programma beheert u zowel TrueType- als PostScript Type 1-lettertypen, zodat u:
- lettertypen onder Windows 98, Windows 2000 Windows XP en Windows Vista kunt installeren en verwijderen 
- nieuwe lettertypen van distributiemedia kunt laden
- op lokale en netwerkstations naar lettertypen kunt zoeken 
- de lettertypen op uw computer of netwerk kunt bekijken
- lettertypegroepen kunt maken voor persoonlijke of werkgroepprojecten
- lettertypen op verschillende manieren kunt sorteren
- ... en nog veel meer!

2.0 SYSTEEMEISEN
Versie van MS Windows: Windows 98, Windows 2000, Windows XP of Windows Vista

Alleen versies van Windows voor één eindgebruiker/werkstation worden ondersteund. Serverversies van Windows worden niet ondersteund.
    
3.0 INSTALLATIE
Plaats de cd-rom in het cd-romstation. De functie voor automatisch starten is ingeschakeld op de cd-rom. Het installatieprogramma zou dus automatisch moeten starten na het plaatsen van de cd.

Selecteer anders Start, Uitvoeren. Voer vervolgens "E:\Setup" in (waarbij E de aanduiding van het cd-romstation is) en druk op Enter. Raadpleeg de Help als u vragen hebt over de installatie. Dubbelklik na het installeren op het pictogram om het programma te starten. Font Manager beschikt over een uitgebreide online Help (druk op F1).

4.0 WERKBALK
Font Manager heeft een werkbalk waarmee u snel toegang hebt tot bewerkingen. Elk pictogram vertegenwoordigt een basisbewerking. Raadpleeg de online Help voor uitleg over de functies van de pictogrammen op de werkbalk van Font Manager.

5.0 MENU'S
Via de menu's van Font Manager hebt u snel toegang tot de basisbewerkingen.

6.0 BESCHIKBARE LETTERTYPEN
Font Manager heeft drie basisvensters. In het eerste staat een lijst van beschikbare lettertypen. In deze lijst kunt u lettertypen selecteren die u in Windows wilt installeren. U kunt ook lettertypen in deze lijst selecteren die u op het scherm wilt bekijken of waarvan u afdrukvoorbeelden wilt hebben.

7.0 GEÏNSTALLEERDE LETTERTYPEN
Het venster met geïnstalleerde lettertypen geeft de familienaam weer van de lettertypen die zijn geïnstalleerd en die voor Windows-toepassingen beschikbaar zijn. Deze lettertypen zijn actief in Windows en kunnen in Windows-toepassingen worden gebruikt.

8.0 LETTERTYPEGROEPEN
Het groeperen van lettertypen is een handige functie in Font Manager. Hiermee kunt u aangepaste verzamelingen lettertypen maken en deze makkelijk herkenbare namen geven.

9.0 SCHERMLETTERTYPEN
Naast Font Manager staat op de distributiemedia ook een set TrueType-schermlettertypen die geschikt zijn voor uw nieuwe printer. Deze TrueType-schermlettertypen zorgen voor een uitstekende WYSIWYG-koppeling tussen uw Windows-toepassingen en uw printer. Uw printer beschikt over deze lettertypen in Monotype Imaging MicroType-indeling, een gepatenteerde lettertypecompressietechnologie.
 
U kunt tijdens het installeren van Font Manager desgewenst de bijpassende TrueType-lettertypen op uw systeem installeren. U kunt ook later nieuwe lettertypen installeren of lettertypen opnieuw installeren.

10.0 TIPS VOOR FONT MANAGER

10.1 LETTERTYPEN ZOEKEN
In "Lettertypen zoeken" van de functie "Stations selecteren" in Font Manager wordt een zoekvenster weergegeven. In dit venster ziet u een pictogram voor het type station, een stationsaanduiding en de stationsclassificatie Lokaal of LAN. Lokaal betekent dat het station zich op de pc bevindt. LAN betekent dat het station zich op een netwerkserver bevindt. In de laatste kolom staat de naam van het station. Deze naam is blanco of een naam die u mogelijk niets zegt (bijv. - MS-DOS 60). De naam is de aanduiding van het volume van het betreffende station, waaraan de Windows-software dit station herkent. Als u naar lettertypen wilt gaan zoeken, klikt u op het station of de stations waarop u wilt zoeken en vervolgens op OK.

10.2 LETTERTYPEGROEPEN
U markeert de naam van een groep en de lettertypen hiervan en heft de markering op door op de lettertypegroepnaam te klikken. Als u een lettertype wilt weergeven, dubbelklikt u hierop in het voorbeeldvenster.

11.0 PROBLEMEN MET FONT MANAGER

11.1 INSTALLATIE VAN HET PROGRAMMA
11.1.1 Als u Font Manager wilt installeren onder Windows Vista of de laatste versie van Windows XP (SP 2), moet u zich hebben aangemeld met beheerdersbevoegdheden (Administrator-bevoegdheden). Tijdens het installatieproces wijzigt het installatieprogramma van Font Manager de toegangsrechten voor bepaalde systeemmappen en registersleutels (zodat Font Manager kan worden uitgevoerd door een gewone gebruiker, zonder beheerdersbevoegdheden).

11.1.2 Tijdens het installatieproces van Font Manager wordt een oudere versie van een Microsoft Access-databasecomponent geïnstalleerd. Deze wordt exclusief gebruikt door Font Manager en dient om de database met geïnstalleerde lettertypen (een .MDB-bestand) te onderhouden. Het installeren van deze oude component heeft geen gevolgen voor de werking van een andere versie van Access die eventueel op uw systeem aanwezig is.

Open echter nooit de Font Manager-database handmatig met Microsoft Access: hierdoor kan de Font Manager-database onherstelbaar beschadigd raken. Mocht u de database per ongeluk op deze manier openen, sla deze dan niet op.

11.3 LETTERTYPEN ZOEKEN
11.3.1 Wanneer er op een diskette wordt gezocht zonder dat een diskette is geplaatst, geeft Font Manager in een foutbericht aan dat er geen diskette in het station aanwezig is. Font Manager begint met de gevraagde zoekactie wanneer de gebruiker het juiste medium heeft geplaatst en op OK heeft geklikt in het dialoogvenster met het foutbericht.

11.3.2 Als u tijdens het zoeken naar lettertypen op een netwerk, dvd-rom of cd-rom op Stop klikt, heeft dit niet onmiddellijk resultaat. U kunt het beste op de knop blijven klikken.

11.3.3 Font Manager probeert te bepalen welk medium (dvd-rom, cd-rom of diskette) in de pc is geplaatst. Als gevolg hiervan ziet u mogelijk de lampjes van de betreffende stations kort oplichten.

11.3.4 Tijdens het zoeken op een netwerk verschijnt mogelijk kort het pad naar de map wanneer bepaalde paden worden doorzocht. Ook is het mogelijk dat wanneer een lettertype wordt gevonden, alleen de naam van het lettertype (zonder het pad naar de map) wordt weergegeven. Font Manager keert terug naar uw bedieningsinterface wanneer de zoekactie is voltooid.

11.3.5 Als u het vak Bewerken gebruikt om het zoekpad op te geven (d.w.z. u typt het pad handmatig in plaats van de bladerfunctie van Windows te gebruiken), moet u onthouden dat het zoeken altijd hoofdlettergevoelig wordt uitgevoerd. Dit geldt zowel voor besturingssystemen die altijd hoofdlettergevoelig zijn (zoals Windows Vista) als voor besturingssysteem die normaal gesproken niet hoofdlettergevoelig zijn (zoals Windows XP). Als u geen zekerheid hebt over eventuele hoofdletters, is het raadzaam de functie Bladeren te gebruiken om het zoekpad op te geven.

11.4 LETTERTYPEN INSTALLEREN
11.4.1 Font Manager ondersteunt momenteel niet de volgende indelingen:
- de MultiMaster-variant van Type 1-lettertypen
- de TrueType Collection-indeling (xxx.TTC)
- de OpenType-indeling (xxx.OTF)
- elke bitmaplettertype-indeling
- de Vista CompositeFont-indeling (xxx.CompositeFont)

11.5 INSTALLATIE VAN LETTERTYPEN ONGEDAAN MAKEN
11.5.1 Font Manager en "Font Set/Auto Activation" van ATM 4.0 veroorzaken een probleem. Wanneer u met Font Manager een Type 1-lettertype verwijdert, wordt dit door ATM automatisch weer geïnstalleerd wanneer Windows opnieuw wordt gestart. Dit is het gevolg van de geavanceerde instelling van ATM 4.0 die "Type 1 Font Controls/Enable Auto-Activation" wordt genoemd. De standaardinstelling van deze functie is "AAN" en dient op "UIT" te worden gezet.

Dit speelt geen rol bij Windows Vista, omdat daarbij geen gebruik van ATM (Adobe Type Manager) meer wordt gemaakt voor het beheer van Type 1-lettertypen.

11.7 LETTERTYPEN WEERGEVEN
11.7.1 De voorbeeldweergave voor bepaalde vetgedrukte of cursieve lettertypetekens wordt op het scherm afgesneden. Als u een kleinere puntgrootte selecteert, verdwijnt het probleem gewoonlijk.

11.7.2 Als u een voorbeeld wilt bekijken van een geïnstalleerde lettertypefamilie met vier stijlen en u een grote puntgrootte selecteert (bijv. - 200 pt), wordt het voorbeeldvenster vergroot, maar kunnen niet alle stijlen worden weergegeven omdat deze aan de onderkant van het scherm buiten het beeld vallen. Selecteer een kleinere puntgrootte waarbij alle stijlen wel in het voorbeeldvenster te zien zijn.

11.10 ALGEMEEN
11.10.1 Font Manager moet niet worden gebruikt voordat het lettertypeconfiguratiescherm van Windows gesloten is.


11.10.2 Wanneer het programma Font Manager er lang over doet om een bepaalde bewerking uit te voeren, kunnen de lijsten "Lettertypen beschikbaar voor installatie" en "Lettertypen geïnstalleerd in Windows" korte tijd verdwijnen.

 Het venster keert terug nadat de betreffende bewerking is voltooid.
