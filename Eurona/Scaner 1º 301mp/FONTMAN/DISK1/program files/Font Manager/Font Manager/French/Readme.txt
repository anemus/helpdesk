1.0 INTRODUCTION
Les polices vous aident à exprimer vos sentiments et à donner du sens à vos documents. Font Manager aide à gérer les polices sous MS Windows. Ce programme gère à la fois les polices TrueType et PostScript Type 1 et vous permet de : 
- installer et désinstaller des polices sous Windows 98, Windows 2000, 
Windows XP et Windows Vista ; 
- charger de nouvelles polices à partir de disques ;
- rechercher des polices sur des unités locales et des unités en réseau ; 
- prévisualiser les polices de votre ordinateur ou du réseau ;
- créer des groupes de polices pour des projets personnels ou des projets de groupe de travail ;
- trier les polices de diverses façons ;
- ... et plus encore !

2.0 CONFIGURATION SYSTÈME REQUISE
Version MS Windows : Windows 98, Windows 2000, Windows XP ou Windows Vista

Seules les versions de Windows pour station de travail/utilisateur final unique sont prises en charge. Aucune version serveur de Windows n’est prise en charge.
    
3.0 INSTALLATION
Insérez le CD dans votre lecteur de CD-ROM. La fonction Exécution automatique est activée sur le CD-ROM. L’installation devrait donc débuter automatiquement après l’insertion du CD.

Dans le cas contraire, sélectionnez Démarrer->Exécuter. Entrez ensuite « E:\Setup » et appuyez sur Entrée (votre lecteur de CD-ROM indique peut-être une autre lettre).Pour toute autre question relative à l'installation, consultez l'aide. Après l'installation, double-cliquez sur l'icône du programme pour lancer celui-ci. Font Manager contient une aide en ligne très complète (appuyez sur F1).

4.0 BARRE D’OUTILS
La barre d’outils de Font Manager contient des icônes permettant d'accéder rapidement aux options. Chaque icône représente une fonction de base. Reportez-vous à l’aide en ligne pour une explication de toutes les icônes de la barre d’outils de Font Manager. 

5.0 MENUS
Les menus de Font Manager permettent d'accéder rapidement aux options.

6.0 POLICES DISPONIBLES
L’une des trois fenêtres de Font Manager présente les polices disponibles. Dans cette liste, sélectionnez les polices à installer dans Windows ou à prévisualiser sur votre écran ou votre imprimante.

7.0 POLICES INSTALLÉES
La fenêtre des polices installées présente le nom de la famille de polices installées et disponibles pour les applications Windows. Ces polices sont disponibles et utilisables dans les applications Windows.

8.0 GROUPES DE POLICES
Groupes de polices est une fonction puissante de Font Manager. Elle vous permet de regrouper les polices au sein d'ensembles personnalisés que vous pouvez nommer.

9.0 POLICES D’ÉCRAN
Votre disque contient des polices d’écran appropriées à votre imprimante au format Monotype Imaging Micro Type, une technologie brevetée de compression des polices. Ces polices peuvent être installées sous la forme TrueType à l’installation du programme, de manière à fournir une excellente fonctionnalité WYSIWYG. Vous pouvez également installer de nouvelles polices ou en réinstaller ultérieurement. 

10.0 CONSEILS FONT MANAGER

10.1 RECHERCHE DE POLICE
Dans « Recherche des polices », sous la fonction « Sélectionner les lecteurs », Font Manager affiche une boîte de dialogue, qui comprend une icône de type de lecteur, une lettre de lecteur et une classification de lecteur (Local ou LAN). Local signifie que l’unité se trouve sur votre PC, LAN sur un serveur du réseau. La dernière colonne affiche le libellé du lecteur. Ce libellé est vierge ou contient un nom qui vous est inconnu (par exemple MS-DOS 60). C’est le libellé du volume du lecteur reconnu par le logiciel Windows. Pour lancer une recherche de polices, cliquez sur le ou les lecteurs souhaités, puis sur le bouton OK.

10.2 GROUPES DE POLICES
Cliquez sur le nom d'un groupe de polices pour mettre en surbrillance ou désélectionner le nom et les polices du groupe. Double-cliquez sur le nom d'une police dans la fenêtre de prévisualisation pour afficher la police.

11.0 PROBLÈMES FONT MANAGER

11.1 PROBLÈMES D'INSTALLATION DE L’APPLICATION
11.1.1 Afin d’installer Font Manager dans l’application Windows Vista ou la dernière version de Windows XP (SP 2), vous devez être connecté avec les privilèges d’administrateur. Dans le cadre de la procédure d’installation, le programme d’installation de Font Manager modifie les permissions d’accès pour certains répertoires système et clés du registre (afin que tout utilisateur non Administrateur puisse accéder à Font Manager).

11.1.2 Une plus ancienne version d’un composant de la base de données Microsoft Access est installée par la procédure d’installation de Font Manager.  Ce composant est uniquement utilisé par Font Manager afin de maintenir sa base de données de polices installées (un fichier .MDB). L’installation de ces composants hérités n’a aucun effet sur les opérations de toutes les autres versions Access se trouvant sur votre système.

Cependant, vous ne devriez jamais ouvrir la base de données Font Manager manuellement avec Microsoft Access. Votre base de données Font Manager pourrait devenir inutilisable. En particulier, vous ne devriez pas enregistrer la base de données si vous l’ouvrez accidentellement.

11.3 RECHERCHE DE PROBLÈME DE POLICE
11.3.1 Dans le cas d’une recherche sur disquette, si aucune disquette n’est insérée dans le lecteur, Font Manager indique une erreur expliquant que le lecteur ne contient pas de disquette. Le programme Font Manager débute la recherche demandée au moment où l’utilisateur insère la disquette appropriée et clique « OK » sur la fenêtre de dialogue d’erreur.

11.3.2 Cliquer sur le bouton Arrêt ne produit aucun effet immédiat lorsque vous recherchez des polices sur le réseau, un DVD-ROM ou un CD-ROM. Il peut s’avérer nécessaire de cliquer sur le bouton plusieurs fois de suite. 

11.3.3. Font Manager essaie de déterminer le type de support (DVD-ROM, CD-ROM ou disquette) inséré dans votre PC. Les voyants correspondant peuvent donc s’allumer brièvement.

11.3.4 Au cours d'une recherche sur le réseau, le chemin du répertoire peut disparaître brièvement ou, lorsqu'une police est trouvée, son nom peut apparaître sans chemin associé. Votre contrôle sur Font Manager est restauré après la recherche.

11.3.5 Si vous ouvrez la zone d’édition pour préciser le chemin d’accès (par ex. vous entrez le chemin d'accès manuellement au lieu d'utiliser la fenêtre du navigateur Windows), votre recherche respecte toujours la casse.  Cela s’applique aussi bien aux OS qui respectent toujours la casse (comme Windows Vista) qu’à ceux qui ne respectent en général pas la casse (comme Windows XP). Si vous n’êtes pas certain de la casse, utilisez la fonction Parcourir pour préciser votre chemin d'accès.

11.4 PROBLÈMES D’INSTALLATION DE POLICE
11.4.1 Font Manager ne prend pas encore en charge les formats suivants :
- la variante MultiMaster des polices de type 1 ;
- l’ensemble de formats TrueType (xxx.TTC) ;
- le format OpenType (xxx.OTF) ;
- tous les formats de police bitmap ;
- le format Vista CompositeFont (xxx. CompositeFont).

11.5 PROBLÈMES DE DÉSINSTALLATION DE POLICE
11.5.1 Font Manager et la commande « Définition de police/Activation automatique » d’ATM 4.0 posent un problème. Lorsque Font Manager désinstalle une police de type 1, ATM la réinstalle automatiquement au redémarrage de Windows. Cela est dû au paramètre avancé d’ATM 4.0 dénommé « Contrôles polices Type 1/Activation automatique ». Ce paramètre avancé est actif et doit être désactivé.

Comme Windows Vista n'utilise plus ATM (Adobe Type Manager) pour gérer les polices de type 1, le problème y est absent.

11.7 PRÉVISUALISATION DES PROBLÈMES DE POLICE
11.7.1 La prévisualisation à l'écran de certaines polices en gras ou en italique peut être tronquée. Essayez de sélectionner une taille inférieure et le problème devrait disparaît.

11.7.2. Lors de la prévisualisation d’une police offrant quatre styles, si vous sélectionnez une grande taille (par ex., 200 points), la fenêtre de prévisualisation s'agrandit, mais certains styles risquent de défiler au delà du bas de la fenêtre. Réduisez simplement la taille jusqu’à ce que tous les styles deviennent visibles.
