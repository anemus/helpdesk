1.0 BEM-VINDO AO FONT MANAGER
As fontes são recursos que ajudam a transmitir sentimentos e significados aos textos e apresentações. O Font Manager auxilia no gerenciamento de fontes no MS Windows. Por suportar as fontes TrueType e Postscript Type 1, ele fornece um único e abrangente programa que permite:
- Instalar e desinstalar fontes no Windows 98, 2000, 
	XP e Vista
- Carregar novas fontes a partir de mídias de distribuição
- Localizar fontes em unidades locais e da rede
- Visualizar fontes locais ou da rede
- Criar grupos de fontes para projetos individuais ou em grupo
- Classificar fontes de diferentes maneiras
- ... e muito mais!

2.0 REQUISITOS MÍNIMOS DO SISTEMA
Versão do MS Windows: Windows 98, Windows 2000, Windows XP ou Windows Vista

São suportadas apenas versões do Windows para um único usuário final/estação de trabalho. Não são suportadas versões do Windows para servidores.
    
3.0 INSTALAÇÃO
Para instalar o programa insira o CD na unidade de CD-ROM. O recurso Reprodução automática está ativado no CD-ROM, portanto, o instalador deverá ser iniciado automaticamente após o CD ser inserido.

Se isso não ocorrer, clique no botão Iniciar, selecione a opção Executar, digite "E:\Setup" (sua unidade de CD-ROM pode ter uma letra diferente) e pressione ENTER. Se tiver dúvidas durante a instalação, consulte a Ajuda on-line do programa. Após a instalação do programa, clique duas vezes no ícone do programa para iniciá-lo. O Font Manager inclui um abrangente sistema de ajuda on-line. Para acessá-lo, basta pressionar a tecla F1.

4.0 BARRA DE FERRAMENTAS
O Font Manager possui uma barra de ferramentas para agilizar o acesso às operações. Cada ícone representa uma das principais operações. Consulte o sistema de ajuda on-line para obter uma explicação das funções de cada ícone da barra de ferramentas do Font Manager.

5.0 MENUS
As opções de menu permitem acesso rápido às principais operações do Font Manager.

6.0 FONTES DISPONÍVEIS
Há três janelas principais exibidas no Font Manager. A primeira é referente à lista de fontes disponíveis. É nessa lista que você seleciona fontes para instalar no Windows. Você também pode selecionar fontes dessa lista para visualizar na tela ou para obter amostras de impressão.

7.0 FONTES INSTALADAS
A janela de fontes instaladas exibe o nome da família de fontes que estão instaladas e ativas no Windows. Essas fontes estão disponíveis no Windows e podem ser utilizadas em seus aplicativos.

8.0 GRUPOS DE FONTES
O agrupamento de fontes é um poderoso recurso do Font Manager. Ele permite que você crie coleções personalizadas de fontes e forneça nomes para identificá-las.

9.0 FONTES DE TELA
Além do Font Manager, a mídia de distribuição também possui um conjunto de fontes de tela TrueType que são compatíveis com sua nova impressora. Essas fontes de tela TrueType foram selecionadas para fornecer uma excelente conexão WYSIWYG (imagem da tela igual ao resultado impresso) entre os aplicativos do Windows e a impressora. A impressora contém essas fontes no formato Monotype Imaging’s MicroType, uma tecnologia patenteada de alta compactação de fontes.
 
Você pode instalar as fontes TrueType compatíveis no seu sistema durante a instalação do programa Font Manager. Você também pode instalar novas fontes ou reinstalar fontes posteriormente.

10.0 DICAS SOBRE O FONT MANAGER

10.1 LOCALIZAÇÃO DE FONTES
Clique em "Selecionar unidades" > "Localizar fontes" para exibir uma janela de busca no Font Manager. Nessa janela, serão exibidos o ícone do tipo da unidade, a letra correspondente à unidade e a classificação da unidade como Local ou LAN. A classificação "Local" significa que a unidade está no seu computador, enquanto a classificação "LAN" significa que a unidade está em um servidor de rede. A última coluna exibe a descrição da unidade. Essa descrição está em branco ou contém um nome que pode não ser familiar a você (ex.: - MS-DOS 60). Esse nome é a descrição do volume dessa unidade reconhecida pelo software do Windows. Para dar início à Localização de fontes, clique nas unidades que você deseja pesquisar e no botão "OK".

10.2 GRUPOS DE FONTES
Para realçar ou remover o realce de um grupo e suas respectivas fontes, basta clicar no nome do Grupo de fontes. Clique duas vezes no nome de uma fonte para abrir a janela de visualização com a exibição da fonte.

11.0 SOBRE O FONT MANAGER

11.1 INSTALAÇÃO DO PROGRAMA
11.1.1 Para instalar o Font Manager no Windows Vista ou na última versão do Windows XP (SP 2), você precisa estar conectado com privilégios de Administrador. Como parte do processo de instalação, o instalador do Font Manager modificará as permissões de acesso para alguns diretórios do sistema e chaves de registro (para que o Font Manager possa ser executado por um usuário comum sem privilégios de Administrador).

11.1.2 Durante a instalação do Font Manager, uma versão mais antiga de um componente do banco de dados do Microsoft Access será instalada. Esse componente é utilizado exclusivamente pelo Font Manager para preservar o banco de dados de fontes instaladas (um arquivo .MDB). A instalação desse componente mais antigo não vai afetar a operação de outra versão do Access disponível no seu sistema.

Entretanto, nunca abra manualmente o arquivo do banco de dados do Font Manager com o Microsoft Access, pois isso pode corromper o banco de dados e inutilizá-lo. Excepcionalmente, se o banco de dados for aberto por acidente, ele não deve ser salvo.

11.3 LOCALIZAÇÃO DE FONTES
11.3.1 Quando uma busca é iniciada em uma unidade de disquete vazia, o Font Manager exibe uma mensagem de erro informando que a unidade está vazia. O programa Font Manager inicia a busca solicitada após o usuário inserir a mídia adequada e clicar em "OK" na janela da mensagem de erro.

11.3.2 Se você clicar no botão "Parar" durante a busca de fontes em uma rede, unidade de DVD-ROM ou de CD-ROM, a busca não será cancelada imediatamente. Para cancelar a busca, pode ser necessário clicar várias vezes nesse botão.

11.3.3 O Font Manager tenta definir qual mídia (DVD-ROM, CD-ROM ou disquete) foi inserida no seu computador. Por isso, as luzes que indicam o funcionamento das unidades de disco poderão se acender por alguns instantes.

11.3.4 Durante uma busca na rede, o caminho do diretório pode desaparecer por alguns instantes enquanto o sistema pesquisa determinados caminhos. Além disso, quando uma fonte é localizada, apenas o nome da fonte é exibido, e não o caminho do diretório.  Confirme se o Font Manager retorna para o seu controle após o término da busca.

11.3.5 A busca sempre faz distinção entre letras maiúsculas e minúsculas quando a caixa de diálogo Editar é utilizada para especificar o caminho da busca (isto é, quando você digita manualmente o caminho, em vez de utilizar o recurso de Pesquisa do Windows). Isso ocorre tanto nos sistemas operacionais que sempre fazem essa distinção (como o Windows Vista) quanto nos que normalmente não fazem (como o Windows XP). Se tiver dúvidas quanto ao seu sistema operacional, recomendamos que você use o recurso de Pesquisa para especificar o caminho da busca.

11.4 INSTALAÇÃO DE FONTES
11.4.1 O Font Manager não suporta atualmente os seguintes formatos:
- variante MultiMaster das fontes Type 1
- formato TrueType Collection (xxx.TTC)
- formato OpenType (xxx.OTF)
- qualquer formato de fonte bitmap
- formato Vista CompositeFont (xxx.CompositeFont)

11.5 DESINSTALAÇÃO DE FONTES
11.5.1 O recurso "Font Set/Auto Activation" (Definição de fontes/Auto-ativação) do ATM 4.0 apresenta um problema quando usado com o Font Manager. Quando o Font Manager desinstala uma fonte Type 1, o ATM a reinstala automaticamente quando o Windows é reiniciado. Isso ocorre devido à configuração avançada "Controles das fontes Type 1/Ativar Auto-ativação" do ATM 4.0. Por padrão, esse recurso está configurado como "ATIVADO" e deve ser alterado para "DESATIVADO".

Observe que isso não se aplica ao Windows Vista, uma vez que ele não utiliza mais o ATM (Adobe Type Manager) para gerenciar fontes Type 1.

11.7 VISUALIZAÇÃO DE FONTES
11.7.1 Alguns caracteres de fontes em Negrito ou Itálico são cortados durante a exibição na tela. Esse problema normalmente é solucionado com a seleção de um tamanho de ponto menor.

11.7.2 Quando você visualiza qualquer família de fonte instalada com quatro estilos e seleciona um tamanho de ponto maior (ex.: 200 pt), a janela de visualização é expandida, mas é incapaz de exibir todos os estilos porque eles ultrapassam o limite da tela. Basta selecionar um tamanho de ponto menor até que todos os estilos sejam exibidos na janela de visualização.

11.10 QUESTÕES GERAIS DO FONT MANAGER
11.10.1 O Font Manager não deve ser utilizado enquanto o Painel de controle de fontes do Windows estiver aberto.

11.10.3 Quando o programa Font Manager demora muito para executar uma determinada operação, as listas "Fontes disponíveis para instalação" e "Fontes instaladas no Windows" podem desaparecer por alguns instantes. As janelas voltarão a ser exibidas quando
a operação for concluída.
