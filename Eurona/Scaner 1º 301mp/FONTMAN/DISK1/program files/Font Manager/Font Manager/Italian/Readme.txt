1.0 INTRODUZIONE A FONT MANAGER
I font sono risorse che conferiscono a testi e presentazioni sensazioni e aspetti estetici particolari. Font Manager consente di gestire in MS Windows i font sia di tipo TrueType che Postscript Type 1, fornendo un unico programma completo, in grado di svolgere le operazioni seguenti:
- Installare e disinstallare font in Windows 98, Windows 2000, 
	Windows XP e Windows Vista
- Caricare nuovi font dai supporti di distribuzione
- Ricercare i font nelle unità locali e di rete
- Visualizzare un'anteprima dei font sul computer locale o in rete
- Creare gruppi di font per progetti personali o di gruppi di lavoro
- Ordinare i font in modi diversi
- ... ... e molto altro ancora.

2.0 REQUISITI MINIMI DI SISTEMA
Versioni di MS Windows: Windows 98, Windows 2000, Windows XP o Windows Vista

Sono supportate esclusivamente le versioni di Windows per un unico utente finale/un'unica workstation. Non sono supportate versioni server di Windows.
    
3.0 INSTALLAZIONE
Per installare il programma, inserire il CD-ROM nell'apposita unità. Sul CD-ROM è attiva la funzione di esecuzione automatica, pertanto l'installazione viene avviata automaticamente una volta inserito il CD.

In caso contrario, dal pulsante Start, selezionare Esegui, quindi digitare "E:\Setup" e premere INVIO (è possibile che l'unità CD-ROM sia contrassegnata da una lettera diversa da E). In caso di incertezze durante l'installazione, fare riferimento alla Guida in linea. Al termine dell'installazione, fare doppio clic sull'icona del programma per avviarlo. Font Manager è corredato da una Guida in linea esauriente che può essere aperta premendo F1.

4.0 BARRA DI CONTROLLO
Font Manager è dotato di una barra di controllo che consente di rendere più rapido l'accesso alle varie operazioni. Ciascuna icona rappresenta un'operazione di base. Per informazioni sulle funzioni di ciascuna icona della barra di controllo di Font Manager, fare riferimento alla Guida in linea.

5.0 MENU
Font Manager consente di accedere in modo rapido alle operazioni di base mediante i menu.

6.0 FONT DISPONIBILI
In Font Manager vengono visualizzate tre finestre di base. La prima finestra comprende l'elenco dei font disponibili. Utilizzare questo elenco per selezionare i font da installare in Windows, per visualizzarne un'anteprima sullo schermo o per ottenerne esempi di stampa.

7.0 FONT INSTALLATI
Nella finestra dei font installati viene visualizzato il nome della famiglia alla quale appartiene ogni font installato e attivo in Windows. Tali font sono attivi in Windows e possono pertanto essere utilizzati per le applicazioni Windows.

8.0 GRUPPI DI FONT
Font Manager dispone di una funzione che consente di raggruppare i font creando collezioni personalizzate di font e assegnando loro i nomi desiderati.

9.0 FONT PER LO SCHERMO
Insieme a Font Manager, il supporto di distribuzione dispone di una serie di font per lo schermo di tipo TrueType corrispondenti a quelli per la nuova stampante. I font TrueType per lo schermo sono stati appositamente selezionati per fornire un collegamento WYSIWYG ottimale tra le applicazioni Windows e la stampante. La stampante include questi font in formato Monotype Imaging's MicroType, una tecnologia brevettata per un'elevata compressione dei font.
 
È possibile installare i font TrueType corrispondenti sul sistema al momento dell'installazione del programma Font Manager. Tuttavia, è sempre possibile installare nuovi font o reinstallare font in momenti successivi.

10.0 SUGGERIMENTI PER L'USO DI FONT MANAGER

10.1 RICERCA DEI FONT
In "Cerca font", sotto la funzione "Seleziona unità", Font Manager visualizza una finestra di ricerca in cui vengono visualizzate l'icona del tipo di unità, la lettera dell'unità e la classificazione dell'unità in base a Locale o LAN. Locale significa che l'unità si trova sul PC locale; LAN significa che l'unità si trova su un server di rete. Nell'ultima colonna viene visualizzato il nome dell'unità. Il campo del nome può essere vuoto oppure può contenere un nome non familiare (ad es. MS-DOS 60), che corrisponde al nome del volume assegnato dal software Windows all'unità. Per avviare la ricerca dei font, fare clic sull'unità/sulle unità in cui si desidera eseguirla, quindi fare clic sul pulsante "OK".

10.2 GRUPPI DI FONT
Facendo clic sul nome di un gruppo di font, il nome del gruppo e i font in esso contenuti vengono evidenziati; facendo di nuovo clic l'evidenziazione viene rimossa. Facendo doppio clic sul nome di un font, appare la finestra di anteprima e viene visualizzato il font.

11.0 PROBLEMI RELATIVI A FONT MANAGER

11.1 PROBLEMI RELATIVI ALL'INSTALLAZIONE DEL PROGRAMMA
11.1.1 Per installare Font Manager in Windows Vista o nell'ultima versione di Windows XP (SP 2), è necessario accedere con diritti di amministratore. Durante il processo di installazione, il programma di installazione di Font Manager modificherà le autorizzazioni di accesso per determinate directory di sistema e chiavi di registro, in modo tale che Font Manager possa essere eseguito da un utente comune non amministratore.

11.1.2 Durante il processo di installazione di Font Manager, viene installata una versione precedente di un componente di Microsoft Access Database. Tale componente viene utilizzato esclusivamente da Font Manager per conservare il database di font installati (file .MDB). L'installazione di questo componente non ha alcun effetto sul funzionamento di qualsiasi versione di Access disponibile sul sistema.

Tuttavia, il file del database di Font Manager non deve essere mai aperto manualmente con Microsoft Access; diversamente, è possibile che il database di Font Manager venga danneggiato e non possa più essere utilizzato. In particolare, se il database viene aperto in modo accidentale, non salvarlo.

11.3 PROBLEMI RELATIVI ALLA RICERCA DEI FONT
11.3.1 Se viene eseguita una ricerca di font sull'unità floppy senza che sia inserito alcun dischetto, Font Manager visualizza un messaggio di errore in cui viene riportato il problema. Una volta che l'utente ha inserito il supporto appropriato e fatto clic su "OK" nella finestra di dialogo di errore, il programma Font Manager avvia la ricerca desiderata.

11.3.2 Durante una ricerca di font in rete, su un DVD-ROM o su un CD-ROM, non è sufficiente fare clic sul pulsante "Ferma" per arrestare la ricerca. A tale scopo, è necessario fare clic sul pulsante più volte.

11.3.3 Font Manager tenta di determinare quale supporto (DVD-ROM, CD-ROM o dischetti floppy) è stato inserito nel PC. Durante tale operazione, è possibile che le spie relative all'unità in questione si accendano per alcuni istanti.

11.3.4 Durante una ricerca in rete, su determinati percorsi, è possibile che il percorso di directory scompaia per alcuni istanti. Inoltre, è possibile che talvolta il nome di un font trovato venga visualizzato senza il relativo percorso di directory. Al termine della ricerca, accertarsi che Font Manager torni alla normale condizione di funzionamento.

11.3.5 Se viene utilizzata la casella Modifica per specificare il percorso di ricerca (ossia, se viene digitato il percorso manualmente e non mediante il browser di Windows), la ricerca effettua la distinzione tra lettere maiuscole e minuscole. Ciò avviene sia in sistemi operativi che effettuano sempre la distinzione tra maiuscole e minuscole, quali Windows Vista, sia in sistemi operativi che generalmente non effettuano tale distinzione, quali Windows XP. In caso di incertezza, utilizzare l'apposita funzione Sfoglia per specificare il percorso di ricerca.

11.4 PROBLEMI RELATIVI ALL'INSTALLAZIONE DEI FONT
11.4.1 Font Manager non supporta attualmente i formati riportati di seguito:
- la variante MultiMaster dei font Type 1
- il formato TrueType Collection (xxx.TTC)
- il formato OpenType (xxx.OTF)
- qualsiasi formato di font bitmap
- il formato Vista CompositeFont (xxx.CompositeFont)

11.5 PROBLEMI RELATIVI ALLA DISINSTALLAZIONE DEI FONT
11.5.1 Un problema è causato da Font Manager e dalla funzione "Imposta font/Attivazione automatica" di ATM 4.0. Quando viene disinstallato un font Type 1 con Font Manager, ATM lo reinstalla automaticamente al nuovo avvio di Windows. Questa operazione è causata da un'impostazione avanzata di ATM 4.0 denominata "Controlli font Type 1/Abilita attivazione automatica". Tale funzione è attiva per impostazione predefinita; per risolvere il problema, è necessario disattivarla.

Questo problema non riguarda Windows Vista, che non utilizza più ATM (Adobe Type Manager) per gestire i font Type 1.

11.7 PROBLEMI RELATIVI ALL'ANTEPRIMA DEI FONT
11.7.1 La visualizzazione dell'anteprima a schermo di alcuni font in grassetto o corsivo può risultare troncata. Per risolvere il problema, generalmente è sufficiente selezionare un corpo più piccolo.

11.7.2 Se viene visualizzata l'anteprima di una famiglia di font installati composta da quattro stili e viene selezionato un corpo più grande (ad es. 200 punti), la finestra dell'anteprima si amplia; tuttavia non sarà possibile visualizzare tutti gli stili in quanto non rientrano all'interno dello schermo. Per visualizzare tutti gli stili all'interno della finestra dell'anteprima, è necessario selezionare un corpo più piccolo.

11.10 PROBLEMI GENERALI RELATIVI A FONT MANAGER
11.10.1 Non utilizzare Font Manager finché il pannello di controllo caratteri di Windows non è chiuso.

11.10.2 Se il programma Font Manager richiede una notevole quantità di tempo per eseguire una data operazione, gli elenchi "Font disponibili per l'istallazione" e "Font installati in Windows" potrebbero scomparire per alcuni istanti. Le finestre con gli elenchi verranno visualizzate di nuovo al termine dell'operazione.
