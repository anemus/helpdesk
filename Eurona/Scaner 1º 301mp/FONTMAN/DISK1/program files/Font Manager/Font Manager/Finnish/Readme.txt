1.0 TERVETULOA KÄYTTÄMÄÄN FONT MANAGERIA
Fontit eli kirjasintyypit auttavat meitä ilmaisemaan dokumenteissa erilaisia tunteita ja merkityksiä Font Manager on MS Windowsissa toimiva fonttien hallinnan apuneuvo. Koska se voi käsitellä sekä TrueType että Postscript tyyppi 1 -fontteja, se on itsenäinen ja kattava ohjelma seuraavien toimenpiteiden suorittamiseen:
- Fonttien asennus ja poisto käyttöjärjestelmissä Windows 98, Windows 2000, Windows XP ja Windows Vista 
- Uusien fonttien lataus jakelutallenteilta
- Fonttien haku paikallisilta ja verkkoasemilta 
- Fonttien esikatselu tietokoneellasi tai verkostasi
- Ryhmäfonttien luominen henkilökohtaisia tai työryhmäprojekteja varten
- Fonttien lajittelu monilla eri tavoilla
- ... ja paljon muuta!

2.0 JÄRJESTELMÄVAATIMUKSET
MS Windows versio: Windows 98, Windows 2000, Windows XP, tai Windows Vista

Vain yhden loppukäyttäjän / työaseman Windows-versioita tuetaan. Windowsin palvelinversioita ei tueta.
    
3.0 ASENNUS
Ohjelman asentamiseksi, ole hyvä ja aseta CD-ROM-levy tietokoneesi CD-asemaan. Mikäli CD-ROM-levyllä on käytössä Automaattinen käynnistys -toiminto, joten asennusohjelman tulisi käynnistyä automaattisesti kun CD työnnetään asemaan.

Muussa tapauksessa, valitse Käynnistä -valikosta Suorita ja syötä "E:\Setup" ja paina ENTER (CD-ROM-asemasi tunnus voi olla eri). Käytönaikainen Ohje opastaa, jos tarvitset neuvoja asennuksen aikana. Kun asennus on valmis, käynnistä ohjelma kaksoisnapsauttamalla sen kuvaketta. Font Manageriin kuuluu osana kattava käytönaikainen ohje (paina F1-painiketta).

4.0 TYÖKALUPALKKI
Font Managerissa on työkalupalkki nopeuttamaan toimintojen käyttöä. Jokainen kuvake vastaa yhtä perustoimintoa. Ole hyvä ja katso jokaisen Font Managerin työkalupalkin kuvakkeen toiminnon kuvaus käytönaikaisesta ohjejärjestelmästä.

5.0 VALIKOT
Font Managerin valikoista pääset kätevästi käsiksi perustoimintoihin toimintoihin.

6.0 KÄYTETTÄVISSÄ OLEVAT FONTIT
Font Managerissa on kolme perusikkunaa. Ensimmäinen näistä listaa käytettävissä olevat fontit. Tästä luettelosta voidaan valita Windowsiin asennettavat fontit. Tästä luettelosta voidaan valita fontteja myös näytöllä esikatseltavaksi tai mallin tulostamista varten.

7.0 ASENNETUT FONTIT
Asennetut fontit -ikkuna näyttää asennettujen ja Windows-sovelluksissa käytössä olevien fonttien perheiden nimet. Nämä fontit ovat Windowsin käytössä ja niitä voidaan käyttää Windows-sovelluksissa.

8.0 FONTTIRYHMÄT
Fonttien ryhmittely on tehokas Font Managerin toiminto. Sen avulla voidaan luoda mukautettuja fonttikokoelmia ja antaa niille kuvaavia nimiä.

9.0 NÄYTTÖFONTIT
Font Managerin lisäksi toimitustallenteella on myös valikoima uudelle tulostimellesi sopivia TrueType-fontteja. Nämä TrueType-näyttöfontit on valittu siten, että ne ovat mahdollisimman tarkasti saman näköisiä Windows-sovelluksissa ja tulosteissasi. Tulostimesi sisältää nämä fontit Monotype Imagingin MicroType-muodossa, joka on patentoitu ja tiukaksi pakattu fonttiteknologia.
 
Voit valita asennetaanko vastaavat TrueType-fontit järjestelmääsi samalla kun asennat Font Manager-ohjelman. Voit tietenkin asentaa vapaasti uusia fontteja tai asentaa samat fontit uudelleen myöhemmin.

10.0 FONT MANAGERIN KÄYTTÖVINKKEJÄ

10.1 FONTTIEN HAKU
Font Manager näyttää hakuikkunan kohdassa "Etsi fontteja -> Valitse asema". Tässä ikkunassa näkyy aseman tyyppikuvake, aseman tunnus ja aseman luokittelu paikalliseksi tai verkkoasemaksi. Paikallinen tarkoittaa että asema on omalla koneellasi, LAN tarkoittaa verkon palvelimeen kytkettyä asemaa. Viimeinen palsta näyttää aseman nimen. Nimikenttä on joko tyhjä tai se sisältää nimen joka ei välttämättä ole sinulle tuttu (esim. MS-DOS 60). Tämä on tietyn Windows-ohjelmiston tunnistaman aseman nimi. Aloita fonttien haku napsauttamalla asemaa(mia) josta(ista) haluat hakea ja napsauta sitten painiketta "OK".

10.2 FONTTIRYHMÄT
Fonttiryhmän nimen napsauttaminen korostaa ryhmään sisältyvien fonttien ja ryhmän nimen tai poistaa korostuksen. Jos kaksoisnapsautat fontin nimeä, Esikatselu-ikkuna avautuu ja näyttää fontin.

11.0 FONT MANAGERIN ONGELMIA

11.1 ASENNUSONGELMIA
11.1.1 Jotta Font Manager voidaan asentaa Windows Vistaan tai Windows XP:n uusimpaan versioon (SP 2), täytyy kirjautua sisään Järjestelmänvalvojan oikeuksin. Asennusmenettelyn osana Font Managerin asennusohjelma muuttaa eräiden järjestelmähakemistojen ja rekisteriavainten pääsyoikeuksia (jotta Font Manageria voi ajaa tavallisen käyttäjän oikeuksin).

11.1.2 Font Managerin asennusmenettelyn osana asennetaan vanhempi versio Microsoft Access Database Component -ohjelmasta. Vain Font Manager käyttää tätä ohjelman osaa oman asennettujen fonttien tietokantansa (.MDB-tiedosto) hallintaan. Tämän vanhan osaohjelman asentamisella ei ole vaikutusta muiden järjestelmässä käytössä olevien Access-versioiden toimintaan.

Sinun ei kuitenkaan koskaan pidä avata Font Managerin tietokantatiedostoa suoraan Microsoft Access -ohjelmalla, sillä se saattaa sotkea Font Managerin tietokannen käyttökelvottomaan kuntoon. Erityisesti jos vahingossa avaat tietokannan, sitä ei tule tallentaa.

11.3 FONTTIEN HAUN ONGELMIA
11.3.1 Haettaessa levykeasemasta jossa ei ole levykettä, Font Manager näyttää virheilmoituksen kertoen ettei asemassa ole levykettä. Font Manager aloittaa pyydetyn haun kun käyttäjä on syöttänyt sopivan tallennusvälineen ja napsauttanut virheilmoitusikkunan painiketta "OK".

11.3.2 Haettaessa fontteja verkosta, DVD-ROM tai CD-ROM -levyiltä, painikkeen "Stop" napsauttamisella ei ole välitöntä vaikutusta. Suositeltava toimenpide on jatkaa painikkeen napsauttelua useita kertoja.

11.3.3 Font Manager yrittää selvittää mitä tallennusvälineitä (DVD-ROM, CD-ROM tai levyke) on PC:n asemissa. Saatat nähdä näiden asemien merkkivalojen syttyvän lyhyesti tästä johtuen.

11.3.4 Verkkohaun kestäessä, muutamaksi hetkeksi hakemistopolku saattaa hävitä hetkellisesti näkyvistä tietyistä poluista haettaessa. Lisäksi, kun fontti on löytynyt, fontin nimi saattaa joskus näkyä ilman hakemistopolkua. Ole huoleti, Font Manager palauttaa hallinnan käyttäjälle kun haku on valmis.

11.3.5 Jos määrität hakupolun Muokkaa-valintarivillä (eli syötät polun käsin etkä käyttäen Windowsin selainta), huomioi hakujen olevan isot ja pienet kirjaimet erotteleva. Tämä pitää paikkansa käyttöjärjestelmille jotka ovat aina isot ja pienet kirjaimet erottelevia (kuten Windows Vista) ja käyttöjärjestelmille jotka eivät yleensä erottele isoja ja pieniä kirjaimia (kuten Windows XP). Jos et ole varma isoista tai pienistä kirjaimista, suosittelemme käyttämään Selaa-toimintoa tarkan hakupolun määrittämiseen.

11.4 FONTTIEN ASENNUSONGELMIA
11.4.1 The Font Manager ei tällä hetkellä tue seuraavia muotoja:
- Type 1 -fonttien MultiMaster-versio 
- TrueType Collection -muoto (xxx.TTC)
- OpenType-muoto (xxx.OTF)
- bittikarttoihin perustuvat fonttimuodot
- Vista CompositeFont -muoto (xxx.CompositeFont)

11.5 FONTTIEN ASENNUKSEN POISTO-ONGELMIA
11.5.1 Font Manager ja ATM 4.0:n "Font Set/Auto Activation" -toiminto aiheuttavat ongelman. Kun Font Manager poistaa Type 1 -fontin, ATM palauttaa asennuksen automaattisesti kun Windows käynnistetään uudelleen. Tämä johtuu ATM 4.0:n lisäasetuksesta nimeltä "Type 1 Font Controls/Enable Auto-Activation." Tämän toiminnon oletusasetus on päällä ("ON") ja se tulee ottaa pois päältä ("OFF").

Tämä ongelma ei enää esiinny Windows Vistassa, joka ei enää käytä ATM-ohjelmaa (Adobe Type Manager) Type 1 -fonttien hallintaan.

11.7 FONTTIEN ESIKATSELUONGELMIA
11.7.1 Joidenkin fonttien lihavoitujen ja kursivoitujen merkkien esikatselukuva leikkautuu ruudun reunoista. Kokeile pienemmän pistekoon valintaan ja ongelma yleensä selviää.

11.7.2 Jos esikatselet jonkin asennetun fonttiperheen neljää tyyliä ja valitset suuren pistekoon (esim. 200 pt), esikatseluikkuna laajentuu, mutta ei kykene esittämään kaikkia tyylejä koska ne vierittyvät näytön alaosasta yli. Valitse vain toinen pienempi pistekoko kunnes kaikki tyylit ovat näkyvissä esikatseluikkunassa.

11.10 FONT MANAGERIN YLEISIÄ ONGELMIA
11.10.1 Font Manageria ei tule käyttää kunnes Windowsin Fontit-ikkuna Ohjauspaneelissa on suljettu.

11.10.2 Kun Font Manager -ohjelmalla kestää pitkän aikaa suorittaa valittu toiminto, "Asennusta varten käytössä olevat fontit" ja "Windowsiin asennetut fonti" -luettelot saattavat kadota näkyvistä lyhyeksi aikaa. Ikkunat palaavat näkyviin kun valittu toiminto on suoritettu.
