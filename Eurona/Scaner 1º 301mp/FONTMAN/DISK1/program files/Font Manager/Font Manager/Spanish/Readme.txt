1.0 BIENVENIDO A FONT MANAGER
Los tipos de letra son recursos que le ayudan a transmitir sentimientos y a reflejar el significado de los documentos. Font Manager le ayuda a gestionar los tipos de letra en Windows. Permite gestionar tipos de letra TrueType y Postscript Tipo 1 mediante un único programa de forma que pueda:
- Instalar y desinstalar tipos de letra en Windows 98, Windows 2000, Windows XP y Windows Vista 
- Cargar nuevos tipos de letra desde diferentes soportes
- Buscar tipos de letra en unidades locales y unidades de red 
- Obtener una vista preliminar de los tipos de letra disponibles en su equipo o en la red
- Crear grupos de tipos de letra para proyectos personales o un grupo de trabajo
- Clasificar los tipos de letra de diferentes de formas
- ... etcétera...

2.0 REQUISITOS MÍNIMOS DEL SISTEMA
Versión de Windows: Windows 98, Windows 2000, Windows XP o Windows Vista

Únicamente se admiten versiones de usuario de equipo local / estación de trabajo de Windows. No se admiten versiones de servidor de Windows.
    
3.0 INSTALACIÓN
Para instalar el programa, introduzca el CD-ROM en la unidad correspondiente. El CD-ROM incluye la función de ejecución automática, por lo que el instalador se debería iniciar automáticamente después de introducir el CD en la unidad.

En caso contrario, en el menú Inicio, seleccione Ejecutar y en el cuadro de texto introduzca "E:\Setup" y pulse INTRO (es posible que su unidad de CD-ROM tenga asignada una letra diferente). La instalación dispone de una ayuda en pantalla, en caso de que tenga dudas durante la instalación. Una vez haya instalado el programa, haga doble clic en el icono del programa para iniciarlo. Font Manager incluye un completo sistema de ayuda en pantalla que puede activar pulsando F1.

4.0 BARRA DE HERRAMIENTAS
Font Manager incluye una barra de herramientas que le permite acceder directamente a las opciones. Cada icono representa una operación básica. Consulte el sistema de ayuda en pantalla para obtener una descripción de las funciones de los iconos de la barra de herramientas de Font Manager.

5.0 MENÚS
Font Manager le permite acceder rápidamente a las operaciones básicas mediante los menús.

6.0 TIPOS DE LETRA DISPONIBLES
Existen tres tipos de ventanas básicas en Font Manager. La primera es la lista de los tipos de letra disponibles. En esta ventana puede seleccionar los tipos de letra que se instalarán en Windows. También puede seleccionar los tipos de letra de los que desea obtener una vista preliminar en pantalla u obtener muestras impresas.

7.0 TIPOS DE LETRA INSTALADOS
La ventana de tipos de letra instalados muestra el nombre de la familia de tipos de letra instalados y que están activos en Windows. Estos tipos de letra están activos y disponibles para las aplicaciones de Windows.

8.0 GRUPOS DE TIPOS DE LETRA
Grupos de tipos de letra es una de las potentes herramientas de Font Manager. Permite crear conjuntos personalizados de tipos de letra y asignarles nombres descriptivos.

9.0 TIPOS DE LETRA EN PANTALLA
Además de Font Manager, el disco también incluye un conjunto de tipos de letra en pantalla TrueType para adecuarlas a su nueva impresora. Estos tipos de letra en pantalla TrueType se han seleccionado con el objeto de proporcionarle una excelente conexión WYSIWYG entre sus aplicaciones de Windows y su impresora. Su impresora contiene estos tipos de letra en el formato MicroType de Monotype Imaging, un sistema patentado de tipos de letra de gran compresión.
 
Puede seleccionar instalar en su sistema los tipos de letra TrueType coincidentes cuando instale Font Manager. Puede instalar nuevos tipos de letra o reinstalarlos más adelante.

10.0 SUGERENCIAS SOBRE FONT MANAGER

10.1 BÚSQUEDA DE TIPOS DE LETRA
Si selecciona la opción "Seleccionar unidades", "Buscar tipos de letra" de Font Manager, se abrirá una ventana de búsqueda. En esta ventana aparecerá un icono del tipo de unidad, la letra de unidad y una clasificación de unidades locales o LAN. Local significa que la unidad se encuentra en su PC, LAN significa que la unidad se encuentra en un servidor de red. La última columna muestra la etiqueta de la unidad. Esta etiqueta puede estar en blanco o contener un nombre que no conozca (por ejemplo, MS-DOS 60). Este nombre es la etiqueta de volumen de una unidad concreta que Windows reconoce. Para iniciar una búsqueda, haga clic en las unidades en la que desee buscar y a continuación haga clic en el botón "Aceptar".

10.2 GRUPOS DE TIPOS DE LETRA
Al hacer clic en un nombre de grupo de tipos de letra se resaltará el nombre del grupo con los tipos de letra que contiene. Si hace doble clic en un nombre de tipo de letra, se abrirá la ventana de vista preliminar y se mostrará el tipo de letra.

11.0 PROBLEMAS DE FONT MANAGER

11.1 PROBLEMAS DE INSTALACIÓN DEL PROGRAMA
11.1.1 Para instalar Font Manager en Windows Vista o la versión más reciente de Windows XP (SP 2), debe iniciar sesión con privilegios de administrador. Como parte del proceso de instalación, el instalador de Font Manager cambiará los permisos de acceso en algunos directorios del sistema y claves de registro (para que un usuario normal que no sea administrador pueda ejecutar Font Manager).

11.1.2 Como parte del proceso de instalación de Font Manager, se instala una versión anterior del componente Microsoft Access Database. Font Manager utiliza este componente exclusivamente para mantener su base de datos de tipos de letra instalados (un archivo .MDB). La instalación de este componente heredado no afecta al funcionamiento de cualquier otra versión de Access que tenga instalada en el sistema.

Sin embargo, no debe abrir manualmente el archivo de base de datos de Font Manager con Microsoft Access, ya que se podría dañar la base de datos de Font Manager y ya no podría volver a utilizarla. De todos modos, si abre accidentalmente la base de datos, no la guarde.

11.3 PROBLEMAS DE BÚSQUEDA DE TIPOS DE LETRA
11.3.1 Si busca en la unidad de disquete sin tener ningún disco insertado, Font Manager muestra un mensaje de error indicando que no hay ningún disco en la unidad. Font Manager iniciará la búsqueda cuando el usuario haya introducido el disco y haga clic en el botón "Aceptar" del cuadro de diálogo de error.

11.3.2 Al buscar tipos de letra en una unidad de red, DVD-ROM o CD-ROM, es posible que si hace clic en el botón "Detener", la búsqueda no se detenga inmediatamente. La mejor opción es pulsar el botón varias veces.

11.3.3 Font Manager intentará determinar el disco (DVD-ROM, CD-ROM o disquete) que ha insertado en su PC. Es posible que se enciendan brevemente las luces de estas unidades.

11.3.4 Cuando realice búsquedas en unidades de red, es posible que la ruta de acceso al directorio desaparezca durante algunos momentos. Cuando se encuentra el tipo de letra, en ocasiones se muestra el nombre del tipo de letra y no la ruta del directorio. Font Manager volverá a estar disponible cuando se complete la búsqueda.

11.3.5 Si utiliza el cuadro de edición para indicar la ruta de búsqueda (por ejemplo, introduciendo la ruta manualmente en lugar de utilizar el explorador de Windows), tenga en cuenta que la búsqueda distinguirá entre mayúsculas y minúsculas. Esta condición se da tanto en sistemas operativos que siempre distinguen entre mayúsculas y minúsculas (como Windows Vista) como en sistemas operativos que normalmente no lo hacen (como Windows XP). Si no sabe de qué tipo se trata, es recomendable que utilice la función de exploración para especificar la ruta de búsqueda.

11.4 PROBLEMAS DE INSTALACIÓN DE TIPOS DE LETRA
11.4.1 Font Manager no admite los siguientes formatos:
- La variante MultiMaster de tipos de letra de Tipo 1
- El formato TrueType Collection (xxx.TTC)
- El formato OpenType (xxx.OTF)
- Cualquier formato de tipo de letra de mapa de bits
- El formato Vista CompositeFont (xxx.CompositeFont)

11.5 PROBLEMAS DE DESINSTALACIÓN DE TIPOS DE LETRA
11.5.1 La función "Conjunto de tipos de letra/Activación automática" de ATM 4.0 y Font Manager causa un problema. Cuando Font Manager desinstala un tipo de letra de Tipo 1, ATM la reinstala automáticamente cuando se vuelve a iniciar Windows. Se debe a la configuración avanzada de ATM 4.0 llamada "Controles de tipos de letra Tipo 1/Habilitar activación automática". Esta configuración está activada por defecto y debe desactivarla.

Tenga en cuenta que este problema no tiene incidencia en Windows Vista, que ya no utiliza ATM (Adobe Type Manager) para gestionar tipos de letra de Tipo 1.

11.7 PROBLEMAS DE VISTA PRELIMINAR DE TIPOS DE LETRA
11.7.1 La vista preliminar en pantalla de determinados tipos de letra en negrita o cursiva puede verse recortada. Este problema se suele solucionar si selecciona un tamaño de punto más pequeño.

11.7.2 Cuando se realiza una vista preliminar de un tipo de letra con cuatro estilos y selecciona un tamaño de punto grande (por ejemplo 200), la ventana de vista preliminar se expande, pero es posible que algunos estilos desaparezcan por la parte inferior de la ventana. Seleccione un tamaño de punto inferior hasta que se muestren todos los estilos en la ventana de vista preliminar.
