1.0 VELKOMMEN TIL FONT MANAGER
Skrifttyper er ressourcer, der er med til at formidle teksters og præsentationers indtryk og betydning.  Font Manager hjælper med at håndtere dine skrifttyper i MS Windows. Da det kan håndtere TrueType og Postscript Type 1, får man et enkelt alsidigt program, hvormed man kan:
- Installere og afinstallere skrifttyper i Windows 98, Windows 2000, 
	Windows XP og Windows Vista 
- Indlæse nye skrifttyper fra distributionsmedier
- Søge efter skrifttyper på det lokale drev og på netværksdrev 
- Vise udskrifter af skrifttyperne på computeren eller netværket
- Oprette skrifttypegrupper til personligt brug eller arbejdsgruppeprojekter
- Sortere skrifttyperne på forskellige måder
- ... og meget mere!

2.0 MINDSTE SYSTEMKRAV
MS Windows Version: Windows 98, Windows 2000, Windows XP eller Windows Vista

Der understøttes kun enkelte Windows slutbruger- / arbejdsstationsversioner. Ingen Windows serverversioner understøttes.
    
3.0 INSTALLATION
Programmet installeres ved at lægge cd-rom’en i cd-drevet. Funktionen Autorun (Kør automatisk) er aktiveret på cd-rom’en, så installationsprogrammet bør starte automatisk, efter cd’en er indsat.

Tryk i modsat fald på knappen Start, vælg Kør og indtast "E:\Setup" og tryk på RETUR (muligvis har dit cd-rom-drev et andet bogstav). Installationen har online hjælp, hvis du har spørgsmål, mens der installeres. Når programmet er installeret, startes det ved at dobbeltklikke på programikonet. Font Manager indeholder et alsidigt online hjælpesystem, der kan hentes frem ved at trykke på tasten F1.

4.0 VÆRKTØJSLINJE
Font Manager har en værktøjslinje, så du kan betjene det hurtigere. Hvert ikon repræsenterer en grundfunktion. Se efter i  det online hjælpesystem for at få forklaret funktionerne af ikonerne på Font Managers værktøjslinje.

5.0 MENUER
Med Font Manager kan grundfunktionerne let åbnes ved at vælge på menuerne.

6.0 SKRIFTTYPER TIL RÅDIGHED
Font Manager viser 3 grundlæggende vinduer. Det første indeholder listen med de skrifttyper, der er til rådighed. Det er fra denne liste, du kan vælge de skrifttyper, der skal installeres i Windows. Du kan også vælge skrifttyper fra listen for at se udskriften på skærmen eller for at se udskriftsprøver. 

7.0 INSTALLEREDE SKRIFTTYPER
Vinduet med installerede skrifttyper viser navnene på de skriftyper, der er installerede og aktive i Windows. Disse skrifttyper er aktive i Windows og kan anvendes i Windows programmer.

8.0 SKRIFTTYPEGRUPPER
Skriftypegruppering er en særdeles nyttig funktion i Font Manager. Den kan bruges til at oprette brugertilpassede skrifttypesamlinger og give dem relevante navne.

9.0 SKÆRMSKRIFTTYPER
Sammen med Font Manager indeholder dine distributionsmedier også et sæt TrueType skærmskrifttyper, der matcher din nye printer. Disse TrueType skærmskrifttyper er blevet valgt, så du får fin overensstemmelse mellem det, du ser på skærmen i dine Windows programmer, og det printeren udskriver. Din printer indeholder disse skrifttyper i Monotype Imagings format kaldet MicroType, en patenteret, stærkt komprimeret skrifttypeteknologi.
 
Du kan vælge at installere de matchende TrueType skrifttyper i systemet, når du installerer programmet Font Manager. Du kan naturligvis altid installere nye skrifttyper eller geninstallere dem på et senere tidspunkt.

10.0 TIPS TIL FONT MANAGER

10.1 SÅDAN SØGES DER EFTER SKRIFTTYPER
I "Search for Fonts" (Søg efter skrifttyper) under funktionen "Select Drives" (Vælg drev) viser Font Manager et søgevindue. I dette vindue ses et ikon for drevtype, et drevbogstav, en drevklassificering som Lokal eller LAN. Lokal betyder, at drevet findes i din egen computer, mens LAN betyder, at drevet ligger på en netværksserver. Den sidste kolonne viser drevets label. Denne label er enten tom eller har et navn, du muligvis ikke kender (dvs.. - MS-DOS 60). Det navn er det bestemte drevs label, som Windows softwaren genkender. For at søge efter skrifttyper, skal du blot klikke på det eller de drev, hvori du ønsker at søge, og så klikke på knappen "OK".

10.2 SKRIFTTYPEGRUPPER
Ved at klikke på skrifttypegruppens navn markerer og fjerner man markeringen af gruppenavnet og de skrifttyper, som gruppen indeholder.  Ved at dobbeltklikke på et skrifttypenavn åbnes vinduet Vis udskrift, hvor skrifttypen kan ses.

11.0 SPØRGSMÅL OM FONT MANAGER 

11.1 SPØRGSMÅL OM PROGRAMINSTALLATIONEN
11.1.1 For at installere Font Manager i Windows Vista eller den nyeste version af Windows XP (SP 2), skal du være logget ind med rettigheder som Administrator. Som en del af installationsprocessen vil Font Managers installationsprogram ændre adgangstilladelserne til bestemte systembiblioteker og registernøgler (så Font Manager kan anvendes af en almindelig bruger, der ikke er Administrator).

11.1.2 Som en del af Font Managers installationsproces installeres en ældre version af en del af Microsoft Access Database. Denne del anvendes udelukkende af Font Manager til at vedligeholde programmets database med installerede skrifttyper (en fil i formatet .MDB). Denne ældre dels installation har ingen indvirkning på driften af nogen anden version af Access, der findes i systemet.

Du bør dog aldrig åbne Font Managers databasefil manuelt via Microsoft Access, da Font Managers database derved kan blive ødelagt, så den ikke længere kan bruges. Husk især, at du ikke må gemme databasen, hvis du er kommet til at åbne den ved et uheld.

11.3 SPØRGSMÅL OM AT SØGE EFTER SKRIFTTYPER
11.3.1 Når der søges på et diskettedrev, og der ikke er nogen diskette i, viser Font Manager en fejldialog, der fortæller, at der ikke er nogen diskette i drevet. Programmet Font Manager begynder den ønskede søgning, når brugeren har indsat det rette medie og klikket på "OK" i vinduet med fejldialogen.

11.3.2 Mens der søges efter skrifttyper enten på et netværk, en dvd-rom eller cd-rom, virker det ikke straks at klikke på knappen "Stop". Det bedste, der kan gøres, er at fortsætte med at klikke mange gange på knappen.

11.3.3 Font Manager forsøger at finde ud af, hvilket medie (dvd-rom, cd-rom eller diskette), der er indsat i computeren. Derfor ser du muligvis, at lamperne til netop de drev lyser i et kort øjeblik.

11.3.4 Mens der søges på et netværk, kan biblioteksstien forsvinde i korte øjeblikke, når der søges i bestemte stier. Nogle gange er det også kun skrifttypens navn, der ses, når skrifttypen er fundet, og ikke biblioteksstien.  Du kan dog styre Font Manager igen, når søgningen er færdig.

11.3.5 Anvender du feltet Rediger til at specificere søgestien (dvs. du indtastet stien manuelt i stedet for at bruge browseren i Windows), skal du huske at skelne mellem store og små bogstaver til søgningen. Det gælder både for operativsystemer, der altid skelner mellem store og små bogstaver (f.eks. Windows Vista), og for operativsystemer, der normalt ikke skelner mellem store og små bogstaver (f.eks. Windows XP). Er du ikke sikker på, hvad der er reglen, anbefaler vi, at du bruger søgefunktionen til at specificere søgestien.

11.4 SPØRGSMÅL OM SKRIFTTYPEINSTALLATION
11.4.1 På nuværende tidspunkt understøtter Font Manager ikke følgende formater:
- varianten MultiMaster af Type 1-skrifttyper
- formatet TrueType Collection (xxx.TTC)
- formatet OpenType (xxx.OTF)
- ethvert skrifttypeformat i bitmap
- formatet Vista CompositeFont (xxx.CompositeFont)

11.5 SPØRGSMÅL OM SKRIFTYPERS AFINSTALLATION
11.5.1 Der opstår et problem mellem Font Manager og ATM 4.0's "Font Set/Auto Activation". Når Font Manager afinstallerer en Type 1-skrifttype, geninstallerer ATM den automatisk, når Windows genstartes. Det skyldes ATM 4.0’s avancerede indstilling kaldet "Type 1 Font Controls/Enable Auto-Activation." Denne funktions standardindstilling er aktiveret, dvs. "ON", og den skal derfor deaktiveres - "OFF."

Bemærk, at det er ikke et problem i Windows Vista, som ikke længere bruger ATM (Adobe Type Manager) til at håndtere Type 1-skrifttyper.

11.7 SPØRGSMÅL OM AT VISE UDSKRIFT AF SKRIFTTYPER
11.7.1 Vis udskift viser nogle skrifttyper i fed eller kursiv som afklippede på skærmen. Prøv at vælge en anden mindre punktstørrelse, da det som regel får problemet til at forsvinde.

11.7.2 Viser du udskriften af nogen installerede skrifttypeserier med 4 stilarter og vælger stor punktstørrelse (dvs. - 200 pt), udvides vinduet Vis udskrift, men det kan ikke vise alle stilarterne, fordi de ruller af nederst på skærmen. Der skal blot vælges en anden mindre punktstørrelse, indtil alle stilarterne er synlige i vinduet med vis udskift.

11,10 GENERELLE SPØRGSMÅL OM FONT MANAGER 
11.10.1 Font Manager bør ikke anvendes, før Skrifttyper i Windows Kontrolpanel er lukket.

11.10.2 Når programmet Font Manager burger lang tid på at udføre en given funktion, forsvinder listerne "Fonts Available for Installation"(Skrifttyper, der kan installers) og "Fonts Installed in Windows" (Skrifttyper installeret i Windows) muligvis kortvarigt. Vinduerne ses igen, når den givne funktion er færdig.
