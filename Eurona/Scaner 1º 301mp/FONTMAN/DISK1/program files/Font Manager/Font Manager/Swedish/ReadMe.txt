1.0 INTRODUKTION
Teckensnitt bidrar till att förmedla känsla och innebörd i dokument. Font Manager hjälper dig att administrera teckensnitt i MS Windows. Programmet hanterar TrueType- och Postscript Type 1-teckensnitt i ett och samma program  så att du  kan:
- Installera och avinstallera teckensnitt i Windows, 98, Windows 2000, 
	Windows XP och Windows Vista 
- Ladda nya teckensnitt från distributionsmedia
- Söka efter teckensnitt lokalt och på nätverksenheter 
- Förhandsgranska lokala teckensnitt och nätverksteckensnitt
- Skapa teckensnittsgrupper för personliga projekt eller arbetsgruppsprojekt
- Sortera dina teckensnitt på många olika sätt
- ... ... med mera!

2.0 SYSTEMKRAV
MS Windows Version: Windows 98, Windows 2000, Windows XP, eller Windows Vista

Endast Windows-versioner för enskilda slutanvändare/arbetsstationer stöds. Inga Windows-versioner för servrar stöds.
    
3.0 INSTALLATION
Sätt in CD-ROM-skivan i din CD-enhet. Autorun-funktionen är aktiverad på CD-ROM-skivan och installationsprogrammet bör startas automatiskt när CD-skivan matas in.

Annars kan du från Start-knappen markera Kör och skriva "E:\Setup" och tryck på ENTER (din CD-ROM-enhet kan ha en annan bokstav). Om du har frågor under installationen går du till hjälpfunktionen. Efter installationen dubbelklickar du på på programikonen för att starta programmet. I Font Manager ingår en omfattande direkthjälp som kan startas genom att trycka på F1.

4.0 VERKTYGSFÄLT
Font Manager har ett verktygsfält med snabbval för olika alternativ. Varje ikon motsvarar en vanlig funktion. Titta i direkthjälpen för en förklaring av funktionerna för varje ikon på Font Manager-verktygsfältet.

5.0 MENYER
Font Managers menyer ger dig möjlighet att snabbt gå till olika alternativ.

6.0 TILLGÄNGLIGA TECKENSNITT
Font Manager har tre grundfönster. Den första av dessa är listan över tillgängliga teckensnitt. Använd denna lista för att välja de teckensnitt som ska installeras i Windows. Du kan även markera teckensnitt från den här listan för att förhandsgranska på skärmen eller för att få skrivarprover.

7.0 INSTALLERADE TECKENSNITT
Fönstret med installerade teckensnitt visar familjenamnen för de teckensnitt som är installerade och tillgängliga för Windows-program. De här teckensnitten är aktiva i Windows och kan användas i Windows-program.

8.0 TECKNESNITTSGRUPPER
Teckensnittsgruppering är en användbar funktion i Font Manager. Det låter dig skapa egna teckensnittssamlingar och ge dem betydelsefulla namn.

9.0 SKÄRMTECKENSNITT
Tillsammans med Font Manager innehåller ditt distributionsmedium även ett antal TrueType-skärmteckensnitt som stämmer överens med din nya skrivare. Dessa TrueType-skärmteckensnitt har valts för att ge dig en utmärkt WYSIWYG-anslutning mellan dina Windows-applikationer och din skrivare. Din skrivare innehåller dessa teckensnitt i Monotype Imagings Micro Type-format, en patenterad och mycket komprimerad teckensnittsteknik.
 
Du kan välja om du vill installera de överensstämmande TrueType-teckensnitten på ditt system när du installerar programmet Font Manager. Du kan alltid installera nya teckensnitt eller installera om teckensnitt senare.

10.0 SÖKA EFTER TECKENSNITT
I “Sök efter teckensnitt”, under funktionen “Markera enheter” visar Font Manager ett sökningsfönter. I det här fönstret får du se en enhetstypikon, en enhetsbokstav och en enhetsklassifikation för Lokalt eller LAN. Lokalt betyder att enheten finns på din dator, LAN betyder att enheten finns på din nätverksserver. Den sista kolumnen visar enhetsetiketten. Den här etiketten är antingen tom eller innehåller ett namn som kanske inte är bekant för dig (t.ex. MS-DOS 60). Det här namnet är volymetiketten för den enhet som kändes igen av Windows-programmet. För att starta en Sökning efter teckensnitt klickar du på de enheter du vill söka i och sedan på ”OK-knappen”.

10,2 TECKNESNITTSGRUPPER
Om du klickar på teckensnittsgruppens namn markeras och avmarkeras gruppnamnet och de teckensnitt som finns i den gruppen. Om du dubbelklickar på ett teckensnittsnamn blir förhandsgranskningsfönstret synligt och teckensnittet visas.

11.0 INSTALLATION
11.1.1. För att installera Font Manager på Windows Vista eller den senaste versionen av Windows XP (SP2) måste du vara inloggad med administratörsbehörighet. Som en del av installationsprocessen kommer Font Manager att ändra åtkomstbehörighet för vissa systemkataloger och registernycklar (så att Font Manager kan köras av en vanlig användare som inte har administratörsbehörighet).

11.1.2 Som en del av installationsprocessen för Font Manager installeras en äldre version av databaskomponenten Microsoft Access. Komponenten används bara av Font Manager för att upprätthålla dess databas av installerade teckensnitt (en MDB-fil). Installering av den här äldre komponenten påverkar inte funktionen för någon annan version av Access som finns på din dator.

Men du bör aldrig öppna Font Managers databasfil manuellt med Microsoft Access eftersom det kan skada Font Manager-databasen så att den inte längre kan användas. Om du i misstag öppnar databasen är det viktigt att du inte spara den.

11.3 SÖKNING EFTER TECKENSNITT
11.3.1 Om du söker på en diskettenhet men har glömt att sätta i disketten så visar Font Manager ett felmeddelande som säger att det inte finns någon diskett i enheten. Font Manager-programmet startar den beställda sökningen när användaren har satt in lämpligt media och klickat på ”OK” på felmeddelandet.

11.3.2 När du söker efter teckensnitt på ett nätverk, DVD-ROM eller CD-ROM har det ingen omedelbar verkan om du trycker på ”Stopp”. Fortsätt att klicka flera gånger på stopp för att avbryta sökningen.

11.3.3. Font Manager försöker avgöra vilket media (DVD-ROM, CD-ROM eller diskett) som är insatt i din dator. Du kan se lamporna på en viss enhet tändas en kort stund pga detta.

11.3.4 Under en nätverkssökning kan katalogens sökväg försvinna under en kort stund medan vissa sökvägar söks. När ett teckensnitt hittas visas ibland bara teckensnittets namn och inte någon katalogsökväg. Font Manager kommer att återgå till din lokala kontroll när sökningen är klar.

11.3.5 Om du använder redigeringsrutan för att ange sökvägen (t.ex. du skriver in sökvägen manuellt i stället för att använda Windows-bläddraren) kommer sökningen alltid att vara skifteslägeskänslig. Det här gäller både för operativsystem som är skifteslägeskänsliga (såsom Windows Vista) och för operativsystem som normalt inte är skifteslägeskänsliga (såsom Windows XP). Om du är osäker på skiftesläget rekommenderar vi att du använder bläddringsfunktionen för att ange din sökväg.

11.4 INSTALLATION AV TECKENSNITT
11.4.1 Font Manager stöder för närvarande inte följande format:
- MultiMaster-varianten av Typ 1 teckensnitt
- TrueType Collection-formatet (xxx.TTC)
- OpenType-formatet (xxx.OTF)
- alla bitmapp teckensnittsformat
- Vista CompositeFont-formatet (xxx.CompositeFont)

11.5 AVINSTALLATION AV TECKENSNITT
11.5.1 Font Manager och ATM 4.0:s "Font Set/Auto Activation” orsakar ett problem. När Font Manager avinstallerar ett Typ 1-teckensnitt. installerar ATM automatiskt om det när Windows startas om. Det beror på de avancerade inställningarna i ATM 4.0 som kallas “Type 1 Font Controls/Enable Auto-Activation”. Den här funktionens standardinställning är ”PÅ” (ON) och bör stängas "AV” (OFF).

Observera att det inte är ett problem i Windows Vista som inte längre använder ATM (Adobe Type Manager) för att hantera Typ 1-teckensnitt.

11.7 FÖRHANDSGRANSKNING AV TECKENSNITT
11.7.1 Förhandsgranskningen för vissa fetstilteckensnitt och kursiva teckensnitt kan vara avskuren i vissa punktstorlekar. Försök markera en annan, mindre punktstorlek så försvinner vanligtvis problemet.

11.7.2 Om du förhandsgranskar en installerad teckensnittsfamilj med fyra stilar och markerar en stor punktstorlek (t.ex. - 200 pt) förstoras förhandsgranskningsfönstret men det kan inte visa alla stilar eftersom de går utanför skärmens nederkant. Markera bara en annan, mindre punktstorlek tills alla stilar är synliga i föhandsgranskningsfönstret.
