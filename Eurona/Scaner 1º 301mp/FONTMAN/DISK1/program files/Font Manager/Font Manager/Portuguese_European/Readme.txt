1.0 BEM-VINDO AO FONT MANAGER
Os tipos de letra são recursos que o ajudam a transmitir emoções e significado no texto e nas apresentações. O Font Manager ajuda-o a gerir os tipos de letra no MS Windows. Como inclui os tipos TrueType e Postscript Type 1, proporciona um programa abrangente único que lhe permite:
- Instalar e desinstalar tipos de letra no Windows 98, no Windows 2000, 
	Windows XP e Windows Vista 
- Carregar novos tipos de letra, a partir de meios de distribuição
- Pesquisar tipos de letra nas unidades locais e de rede 
- Pré-visualizar os tipos de letra no seu computador ou rede
- Criar grupos de tipos de letra para projectos pessoais ou de grupos de trabalho
- Organizar os tipos de letra de várias maneiras
- ... e muito mais!

2.0 REQUISITOS MÍNIMOS DO SISTEMA
Versão MS Windows: Windows 98, Windows 2000, Windows XP ou Windows Vista

Só as versões de Windows para utilizador final único / estação de trabalho são apoiadas. As versões de servidor do Windows não são apoiadas.
    
3.0 INSTALAÇÃO
Para instalar o programa, insira o CD na unidade de CD-ROM. A opção Autorun está activada no CD, portanto, a instalação deve começar automaticamente depois de inserir o CD.

Como alternativa, clique no botão Start (Iniciar), seleccione Run (Executar) e digite “E:\Setup” e prima Enter (a sua unidade do CD-ROM pode ter uma letra diferente). A instalação dispõe de ajuda online, se tiver dúvidas durante a instalação. Assim que o programa tiver sido instalado, clique duas vezes no ícone do programa para iniciá-lo. O Font Manager inclui um abrangente sistema de ajuda online que pode ser activado com F1.

4.0 BARRA DE FERRAMENTAS
O Font Manager dispõe de uma barra de ferramentas para facilitar o acesso às operações. Cada ícone representa uma operação básica. Por favor, consulte o sistema de ajuda online para uma explicação sobre as funções de cada ícone da barra de ferramentas do Font Manager.

5.0 MENUS
O Font Manager permite-lhe aceder facilmente a operações básicas através dos menus.

6.0 TIPOS DE LETRA DISPONÍVEIS
No Font Manager, há três janelas básicas. A primeira é a lista dos tipos de letra disponíveis. É a partir desta lista que pode seleccionar os tipos de letra que pretende instalar no Windows. Também pode seleccionar tipos de letra da lista para pré-visualizar no ecrã ou para imprimir amostras.

7.0 TIPOS DE LETRA INSTALADOS
A janela dos tipos de letra instalados indica o nome da família dos tipos de letra instalados e activos no Windows. Estes tipos de letra estão activos no Windows e podem ser usados nas suas aplicações.

8.0 GRUPOS DE TIPOS DE LETRA
O agrupamento de tipos de fonte é uma excelente opção do Font Manager. Assim, é possível criar grupos personalizados de tipos de letra e dar-lhes nomes adequados.

9.0 TIPOS DE LETRA PARA O ECRÃ
Juntamente com o Font Manager, o seu meio de distribuição também possui um conjunto de tipos de letra para o ecrã TrueType, compatível com a sua nova impressora. Estes tipos de letra TrueType foram escolhidos para lhe proporcionar uma excelente ligação WYSIWYG entre as aplicações do Windows e a impressora. A sua impressora contém estas no formato Microtype de Monotype Imaging, uma tecnologia de tipos de letra patenteada e altamente compactável.
 
Pode optar por instalar os tipos de letra TrueType compatíveis com o seu sistema, quando instalar o programa Font Manager. É claro que pode sempre instalar tipos de letra novos, ou reinstalar tipos de letra, à posteriori.

10.0 DICAS SOBRE O FONT MANAGER

10.1 PESQUISA DE TIPOS DE LETRA
Ao abrir a opção “Search for Fonts” (“Pesquisa de Tipos de Letra”) e, depois, “Select Drives” (“Seleccionar Unidades”), o Font Manager disponibilizar-lhe-á uma janela de pesquisa. Nesta janela, verá um ícone do tipo da unidade, uma letra correspondente à unidade e uma classificação da unidade como Local ou LAN. Local significa que a unidade se encontra no seu PC, LAN significa que a unidade está num servidor de rede. A última coluna apresenta a identificação da unidade. Esta identificação poderá ser em branco ou conter um nome que pode não lhe ser familiar (isto é, MS-DOS 60). Este nome é a identificação de volume de uma unidade específica, reconhecida pelo software do Windows. Para iniciar uma pesquisa de tipos de letra, basta clicar na(s) unidade(s) que desejar pesquisar e, depois, no botão “OK”.

10.2 GRUPOS DE TIPOS DE LETRA
Ao clicar no nome de um grupo de tipos de letra, destacará ou não o nome do grupo e respectivos tipos de letra. Se clicar duas vezes no nome do tipo de letra, aparecerá uma janela de pré-visualização com o tipo de letra seleccionado.

11.0 QUESTÕES SOBRE O FONT MANAGER

11.1 QUESTÕES SOBRE A INSTALAÇÃO DO PROGRAMA
11.1.1 Para instalar o Font Manager no Windows Vista, ou na última versão do Windows XP (SP 2), tem de se registar com privilégios de Administrador. Como parte do processo de instalação, o installer (programa de instalação) do Font Manager mudará as autorizações de acesso para certas directorias do sistema e chaves de registo (para que o Font Manager possa ser executado por um utilizador normal, sem ser o Administrador).

11.1.2 Como parte do processo de instalação do Font Manager, é instalada uma versão mais antiga de um componente de Microsoft Access Database. Este componente é usado exclusivamente pelo Font Manager, para manter a sua base de dados dos tipos de letra instalados (um ficheiro MDB). A instalação deste componente de arquivo não terá qualquer efeito no funcionamento de qualquer outra versão do Access, que esteja disponível no seu sistema. 

No entanto, nunca deverá abrir manualmente o ficheiro da base de dados do Font Manager com o Microsoft Access, pois isso pode corromper a base de dados do Font Manager e impossibilitar qualquer uso futuro. Especificamente, se abrir acidentalmente a base de dados, não deve “guardá-la”, ao sair.

11.3 QUESTÕES SOBRE A PESQUISA DE TIPOS DE LETRA
11.3.1 Se o Font Manager pesquisar a unidade de disquetes e não encontrar uma lá inserida, aparecerá uma mensagem de erro, alertando para a ausência de disquete na respectiva unidade. O programa Font Manager começa a pesquisa pedida, depois de o utilizador ter inserido o meio adequado e clicado em “OK” na janela da mensagem de erro.

11.3.2 Durante a pesquisa de tipos de letra, quer seja numa rede, DVD-ROM ou CD-ROM, se clicar no botão "Stop", não obterá um efeito imediato. A melhor solução é insistir, clicando várias vezes no botão.

11.3.3 O Font Manager tenta identificar o meio (DVD-ROM, CD-ROM ou disquete) que está inserido no PC. Pode ver a luz destas unidades específicas a acender por breves instantes por causa disso.

11.3.4 Durante uma pesquisa na rede, por breves instantes, a via da directoria pode desaparecer, quando se pesquisar certas vias. Além disso, às vezes, quando é detectado um tipo de letra, só aparece o nome e não a indicação da directoria. Por favor, certifique-se de que o Font Manager retomará o seu controlo, depois de concluída a pesquisa.

11.3.5 Se usar a caixa de “Edit” (“Editar”)para especificar a via da pesquisa (isto é, digita manualmente a via em vez de usar o browser do Windows), não se esqueça que a pesquisa será sempre "case-sensitive” (“sensível”). Isto aplica-se tanto a sistemas operativos que são sempre case-sensitive (como o Windows Vista), como a sistemas operativos que normalmente são case-insensitive (como o Windows XP). Se tiver a certeza de qual é o caso, recomendamos que use a opção do browser para especificar a via de pesquisa.

11.4 QUESTÕES SOBRE A INSTALAÇÃO DE TIPOS DE LETRA
11.4.1 O Font Manager não suporta actualmente os seguintes formatos:
- a variante MultiMaster dos tipos de letra Type 1 (“Tipo 1”)
- o formato do conjunto TrueType (xxx.TTC)
- o formato OpenType (xxx.OTF)
- qualquer formato de tipo de letra bitmap
- o formato do Vista CompositeFont (xxx.CompositeFont)

11.5 QUESTÕES SOBRE A DESINSTALAÇÃO DE TIPOS DE LETRA
11.5.1 O Font Manager e o Font Set/Auto Activation do ATM 4.0 causam um problema. Quando o Font Manager desinstala um tipo de letra Type 1, o ATM instala-o automaticamente, quando o Windows é reiniciado. Isto deve-se a uma configuração avançada do ATM 4.0, designada por Type 1 Font Controls/Enable Auto-Activation. Esta configuração instalada por defeito está activada (“ON") e deve ser desactivada ("OFF").

Note-se que isto não é problemático no Windows Vista que já não usa o ATM (Adobe Type Manager) para gerir os tipos de letra Type 1.

11.7 QUESTÕES SOBRE A PRÉ-VISUALIZAÇÃO DE TIPOS DE LETRA
11.7.1 A pré-visualização de alguns caracteres em negrito e em itálico pode apresentar algumas falhas no ecrã. Tente seleccionar um tamanho de ponto mais pequeno e o problema ficará normalmente resolvido.

11.7.2 Se pré-visualizar qualquer família de tipos de letra com quatro estilos e seleccionar um tamanho de ponto grande (isto é, 200 pontos), a janela de pré-visualização aumenta, mas não consegue apresentar todos os estilos, pois ultrapassam a zona inferior do ecrã. Seleccione simplesmente outro tamanho de ponto mais pequeno até todos os estilos ficarem visíveis na janela de pré-visualização.

a dita operação estiver concluída.
