1.0. FONT MANAGER. ПРИВЕТСТВИЕ
Шрифты являются средствами передачи эмоциональной и смысловой нагрузки текста и презентаций. Font Manager предоставляет возможность управления шрифтами в ОС MS Windows. Благодаря тому, что это ПО может обрабатывать шрифты TrueType и Postscript Type 1, оно представляет собой единую комплексную программу, позволяющую выполнять следующие действия:
- устанавливать и удалять шрифты в ОС Windows 98, Windows 2000, 
	Windows XP и Windows Vista;
- загружать новые шрифты с носителей со шрифтами;
- осуществлять поиск шрифтов на локальных и сетевых дисках;
- предварительно просматривать шрифты на компьютере или по сети;
- создавать группы шрифтов для персональных проектов и проектов рабочей группы;
- сортировать шрифты различными способами;
- ... - и многое другое.

2.0. МИНИМАЛЬНЫЕ СИСТЕМНЫЕ ТРЕБОВАНИЯ
Версия MS Windows: Windows 98, Windows 2000, Windows XP или Windows Vista

Поддерживаются только версии для одного конечного пользователя и рабочей станции. Серверные версии Windows не поддерживаются.
    
3.0. УСТАНОВКА
Для установки программы необходимо поместить компакт-диск в дисковод компакт-дисков. На компакт-диске задействована функция автозапуска, поэтому программа установки должна запуститься автоматически после того, как будет вставлен компакт-диск.

В противном случае в меню «Пуск» необходимо выбрать «Выполнить», набрать «E:\Setup» и нажать клавишу ввода «ENTER» (компакт-диск на компьютере может быть обозначен другой буквой). В программе установки доступна встроенная справка, которой можно воспользоваться при возникновении вопросов во время установки. После завершения установки дважды щелкните значок программы для запуска программы. Font Manager содержит подробную встроенную справочную систему, которую можно запустить нажатием клавиши «F1».

4.0. ПАНЕЛЬ ИНСТРУМЕНТОВ
Для облегчения доступа к операциям Font Manager оснащен панелью инструментов. Каждый значок представляет основную операцию. Описание функций значков панели инструментов Font Manager см. во встроенной справочной системе.

5.0. МЕНЮ
Font Manager обеспечивает удобный доступ к основным операциям посредством пунктов меню.

6.0. ДОСТУПНЫЕ ШРИФТЫ
В Font Manager отображается три основных окна. Первое из них представляет собой список доступных шрифтов. В этом списке можно выбирать шрифты, которые необходимо установить в ОС Windows. Также в этом списке можно выбрать шрифты для предварительного просмотра на экране или для получения распечатанных образцов.

7.0. УСТАНОВЛЕННЫЕ ШРИФТЫ
В окне установленных шрифтов отображаются названия семейств шрифтов, установленных и действующих в ОС Windows. Эти шрифты действуют в ОС Windows, их можно использовать в приложениях Windows.

8.0. ГРУППЫ ШРИФТОВ
Возможность группирования шрифтов является мощной функцией Font Manager. Она позволяет создавать пользовательские наборы шрифтов и присваивать им выразительные названия.

9.0. ЭКРАННЫЕ ШРИФТЫ
Вместе с Font Manager на носителе с программой также имеется набор экранных шрифтов TrueType, соответствующих шрифтам приобретенного принтера. Эти экранные шрифты TrueType выбраны с целью обеспечения точного соответствия в режиме WYSIWYG (что видишь, то и получаешь) шрифтов приложений Windows и принтера. Принтер содержит эти шрифты в формате MicroType компании Monotype Imaging, запатентованной технологии сильно сжатых шрифтов.
 
При установке программы Font Manager можно выбрать установку соответствующих шрифтов TrueType на системе. Всегда можно установить новые или повторно установить имеющиеся шрифты позднее.

10.0. СОВЕТЫ ПО ИСПОЛЬЗОВАНИЮ FONT MANAGER

10.1. ПОИСК ШРИФТОВ
При выборе «Поиск шрифтов» в функции «Выбор дисков» Font Manager отображает окно поиска. В этом окне отображаются значок типа диска, буква диска и классификатор диска – локальный или сетевой. Локальный диск означает, что диск расположен в ПК, сетевой – что диск расположен на сетевом сервере. В последнем столбце отображается метка диска. Эта метка может быть пустой или иметь неизвестное имя (т.е. MS-DOS 60). Это имя является меткой тома данного диска, распознаваемой ПО Windows. Для запуска поиска шрифтов следует щелкнуть значок диска или дисков, в которых следует осуществить поиск, и нажать кнопку «OK».

10.2. ГРУППЫ ШРИФТОВ
При щелчке по имени «Группы шрифтов» имя группы и содержащиеся в этой группе шрифты будут выделены, при повторном щелчке выделение будет снято. При двойном щелчке по имени шрифта отобразится окно «Предварительный просмотр» с отображением этого шрифта.

11.1. ПРОБЛЕМЫ FONT MANAGER

11.1. ПРОБЛЕМЫ УСТАНОВКИ ПРОГРАММЫ
11.1.1. Для установки Font Manager в ОС Windows Vista или новейшей версии ОС Windows XP (SP 2) необходимо войти в систему с правами администратора. Во время установки программа установки Font Manager изменит права доступа для определенных системных папок и ключей реестра с целью обеспечения возможности запуска Font Manager обычным пользователем, не имеющим прав администратора.

11.1.2. В процессе установки Font Manager будет установлена старая версия компонента Microsoft Access Database. Этот компонент используется исключительно Font Manager для управления базой данных установленных шрифтов (файл .MDB). Установка этого унаследованного компонента не повлияет на работу других версий Access, установленных в системе.

Однако не следует открывать файл базы данных Font Manager вручную при помощи Microsoft Access, так как это может повредить базу данных Font Manager и ее дальнейшее использование будет невозможно. В частности, при случайном открытии базы данных никогда не следует ее сохранять.

11,3. ПРОБЛЕМЫ ПОИСКА ШРИФТОВ
11.3.1. При поиске шрифтов на дисководе гибких дисков без вставленной дискеты Font Manager отображает диалоговое окно с сообщением об ошибке, указывая на отсутствие дискеты в дисководе. Font Manager начнет запрошенный поиск после того, как пользователь вставит соответствующий носитель и нажмет кнопку «OK» в диалоговом окне с сообщением об ошибке.

11.3.2. Во время поиска шрифтов в сети, на DVD- или компакт-диске кнопка «Стоп» действует не сразу. Наилучшим решением является повторное нажатие кнопки несколько раз.

11.3.3. Font Manager пытается определить, какой носитель (DVD, компакт-диск или дискета) вставлен в ПК. При этом значок этого конкретного носителя слегка подсвечивается.

11.3.4. Во время поиска по сети на некоторый момент может исчезать путь папки при поиске в определенных каталогах. Также при обнаружении шрифта иногда отображается только его название без пути папки. Помните, что после завершения поиска Font Manager исправно будет выполнять команды пользователя.

11.3.5. При использовании адресной строки для указания пути поиска (т.е. при вводе пути вручную вместо использования проводника Windows) примите во внимание, что поиск учитывает регистр. Это действительно как для ОС, которые всегда учитывают регистр (например, Windows Vista), так и для ОС, которые обычно не учитывают регистр (например, Windows XP). Если вы не уверены в регистре вводимого пути, рекомендуется использовать обзор для указания пути поиска.

11.4. ПРОБЛЕМЫ УСТАНОВКИ ШРИФТОВ
11.4.1. На данный момент Font Manager не поддерживает следующие форматы:
- вариант MultiMaster шрифтов Type 1;
- формат TrueType Collection (xxx.TTC);
- формат OpenType (xxx.OTF);
- все форматы растровых шрифтов
- формат Vista CompositeFont (xxx.CompositeFont).

11.5. ПРОБЛЕМЫ УДАЛЕНИЯ ШРИФТОВ
11.5.1. Возникает проблема при использовании Font Manager и функции «Font Set/Auto Activation» (Набор шрифтов/Автоматическая активация) программы ATM 4.0. Каждый раз при удалении программой Font Manager любого шрифта Type 1 программа ATM автоматически устанавливает его повторно при перезагрузке Windows. Это происходит в связи со значением усовершенствованного параметра «Type 1 Font Controls/Enable Auto-Activation» (Управление шрифтами Type 1/Включить автоматическую активацию). Стандартное значение этого параметра – «ON» (ВКЛ.), его следует изменить на «OFF» (ВЫКЛ.).

Помните, что такой проблемы не возникает в ОС Windows Vista, в которой ATM (Adobe Type Manager) больше не используется для управления шрифтами Type 1.

11.7. ПРОБЛЕМЫ ПРЕДВАРИТЕЛЬНОГО ПРОСМОТРА ШРИФТОВ
11.7.1. При предварительном просмотре некоторые символы полужирных и курсивных шрифтов отображаются на экране в «обрезанном» виде. Обычно такая проблема устраняется путем установки меньшего размера знаков.

11.7.2. При предварительном просмотре неустановленного семейства шрифтов в четырех стилях и выборе большого размера знаков (т.е. от 200 точек) окно предварительного просмотра расширяется, однако при этом невозможно отобразить все стили, т.к. они выходят за нижний край окна. Следует выбрать меньший размер знаков, чтобы все стили отображались в окне предварительного просмотра.

11.10. ОБЩИЕ ПРОБЛЕМЫ FONT MANAGER
11.10.1. Font Manager не следует использовать при открытой панели управления шрифтами Windows.

11.10.2. Если программе Font Manager требуется длительное временя для выполнения заданной операции списки «Доступные для установки шрифты» и «Установленные в Windows шрифты» могут не отображаться в течение короткого времени. Окна будут восстановлены после
завершения выполняемой операции.
