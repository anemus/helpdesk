1.0 EINFÜHRUNG
Mithilfe von Schriftarten können Sie in Texten und Präsentationen Stimmungen und Bedeutungen vermitteln. Der Font Manager unterstützt Sie bei der Verwaltung von Schriftarten unter MS Windows. Das Programm fasst die Behandlung von TrueType und Postscript (Typ 1) in einem Programm zusammen. Sie können somit:
- Schriftarten unter Windows 98, Windows 2000, Windows XP und Windows Vista installieren und deinstallieren
- neue Schriftarten von Verteilungsmedien laden
- lokale Laufwerke und Netzwerklaufwerke nach Schriftarten durchsuchen
- lokal oder im Netzwerk installierte Schriftarten in der Vorschau anzeigen
- Schriftgruppen für den Eigengebrauch oder Arbeitsgruppenprojekte erstellen
- Schriftarten nach unterschiedlichen Kriterien sortieren
- ... und noch viel mehr!

2.0 SYSTEMANFORDERUNGEN
MS Windows-Version: Windows 98, Windows 2000, Windows XP oder Windows Vista

Unterstützt nur Windows-Versionen für Einzelbenutzer/Arbeitsstationen. Windows-Serverversionen werden nicht unterstützt.
    
3.0 INSTALLATION 
Legen Sie zum Installieren des Programms die CD-ROM in das CD-Laufwerk ein. Die CD-ROM wird mit aktivierter Autorun-Funktion geliefert. Das Installationsprogramm startet deshalb nach Einlegen der CD-ROM automatisch.

Falls nicht, wählen Sie "Start" > "Ausführen". Geben Sie anschließend "E:\Setup" ein, und drücken Sie die EINGABETASTE. (Das CD-ROM-Laufwerk Ihres Computers wird möglicherweise durch einen anderen Laufwerksbuchstaben identifiziert.) Falls während der Installation Fragen auftreten, können Sie die Installations-Onlinehilfe aufrufen. Nach der Installation starten Sie das Programm per Doppelklick auf das Programmsymbol. Der Font Manager ist mit einer umfassenden Onlinehilfe ausgestattet. Sie rufen die Hilfe mit der F1-Taste auf.

4.0 SYMBOLLEISTE
Die Symbolleiste des Font Manager ermöglicht einen schnelleren Zugriff auf Operationen. Jedes Symbol steht für eine grundlegende Operation. Erläuterungen zu den Funktionen aus der Font Manager-Symbolleiste finden Sie in der Onlinehilfe.

5.0 MENÜS
Der Font Manager ermöglicht es Ihnen, über entsprechende Menüoptionen bequem grundlegende Operationen aufzurufen.

6.0 VERFÜGBARE SCHRIFTARTEN
Das Font Manager-Fenster setzt sich aus drei Bereichen zusammen. Im ersten Bereich sehen Sie eine Liste mit den verfügbaren Schriftarten. Wenn Sie Schriftarten in Windows installieren möchten, wählen Sie aus dieser Liste. Sie können auch Schriftarten aus dieser Liste auswählen, um eine Bildschirmvorschau anzuzeigen oder eine Druckprobe auszugeben.

7.0 INSTALLIERTE SCHRIFTARTEN
Im Bereich mit den installierten Schriftarten sehen Sie Bezeichnungen der unter Windows installierten und aktiven Schriftarten. Diese Schriftarten sind unter Windows aktiv und können in Windows-Anwendungen eingesetzt werden.

8.0 SCHRIFTARTENGRUPPEN
Die Gruppierung von Schriftarten ist ein herausragendes Merkmal des Font Manager. Sie können Schriftarten individuell zusammenstellen und die Gruppe mit einem aussagekräftigen Namen versehen.

9.0 BILDSCHIRMSCHRIFTARTEN
Auf Ihrem Distributionsmedium sind neben dem Font Manager mehrere TrueType-Bildschirmschriftarten für Ihren neuen Drucker gespeichert. Diese TrueType-Bildschirmschriftarten wurden ausgewählt, weil sie eine hervorragende Darstellungsübereinstimmung zwischen Windows-Anwendung und Ihrem Drucker gewährleisten. Auf Ihrem Drucker sind diese Schriftarten im MicroType-Format von Monotype Imaging hinterlegt, einer patentierten, hochgradig komprimierten Schriftartentechnologie.
 
Sie können die passenden TrueType-Schriftarten zusammen mit dem Font Manager auf Ihrem System installieren. Natürlich können Sie jederzeit neue Schriftarten installieren oder Schriftarten erneut installieren.

10.0 TIPPS ZUM FONT MANAGER

10.1 NACH SCHRIFTARTEN SUCHEN
Unter "Laufwerke auswählen" > "Nach Fonts suchen" sehen Sie ein Suchfenster. Dieses Fester enthält ein Laufwerkssymbol, einen Laufwerksbuchstaben und eine Laufwerksklassifizierung (lokal oder LAN). "lokal" bedeutet, dass sich das Laufwerk auf Ihrem PC befindet, "LAN" bedeutet, dass sich das Laufwerk auf einem Netzwerkserver befindet. In der letzten Spalte ist die Laufwerksbezeichnung vermerkt. Entweder ist die Spalte leer oder enthält eine ungewohnte Bezeichnung (MS-DOS 60). Dieser Name ist die Laufwerksbezeichnung des von Windows erfassten Laufwerks. Um nach Schriftarten zu suchen, klicken Sie einfach auf die Laufwerke, die Sie durchsuchen möchten, und klicken auf OK.

10.2 SCHRIFTARTENGRUPPEN
Wenn Sie auf eine Schriftartengruppe klicken, markieren Sie den Gruppennamen und die Schriftarten der Gruppe bzw. heben die Markierung auf. Wenn Sie auf einen Schriftartennamen doppelklicken, wird das Schriftartenfenster mit der jeweiligen Schriftart angezeigt.

11.0 PROBLEME BEI DER NUTZUNG VON FONT MANAGER

11.1 PROBLEME BEI DER PROGRAMMINSTALLATION
11.1.1 Um den Font Manager unter Windows Vista oder der letzten Version von Windows XP (SP 2) installieren zu können, müssen Sie mit Administratorrechten angemeldet sein. Währen des Installationsvorgangs ändert das Installationsprogramm des Font Manager die Zugriffsberechtigungen für bestimmte Systemverzeichnisse und Registrierungsschlüssel (damit der Font Manager auch von normalen Benutzern ausgeführt werden kann, die nicht Administratorrechte besitzen).

11.1.2 Während der Installation von Font Manager wird eine ältere Version einer Microsoft Access-Datenbankkomponente installiert. Diese Komponente wird ausschließlich durch den Font Manager genutzt, um damit seine Datenbank installierter Schriftarten (.MDB-Datei) zu verwalten. Die Installation dieser Legacy-Komponente hat keinerlei Auswirkungen auf andere Access-Versionen auf Ihrem System.

Öffnen Sie aber nicht die Font Manager-Datenbank manuell mit Microsoft Access. Dadurch könnte die Font Manager-Datenbank beschädigt und unbrauchbar werden. Wenn Sie dennoch aus Versehen die Datenbank geöffnet haben, sollten Sie auf keinen Fall speichern.

11.3 PROBLEME BEI DER SUCHE NACH SCHRIFTARTEN
11.3.1 Wenn ein Floppy-Laufwerk durchsucht wird, in das keine Diskette eingelegt ist, weist Sie eine Font Manager-Fehlermeldung darauf hin, dass keine Diskette in das Laufwerk eingelegt ist. Der Font Manager startet die gewünschte Suche, wenn der Benutzer die entsprechenden Medien eingelegt hat und auf die Schaltfläche "OK" der Fehlermeldung geklickt hat.

11.3.2 Wenn Sie auf einem Netzwerk, einer DVD-ROM oder CD-ROM nach Schriftarten suchen, geschieht nichts, wenn Sie auf "Stopp" klicken. Am besten versuchen Sie es weiterhin, indem Sie mehrmals auf die Schaltfläche klicken.

11.3.3 Der Font Manager versucht festzustellen, welche Medien (DVD-ROM, CD-ROM oder Diskette) in den PC eingesetzt wurden. Deswegen kann es vorkommen, dass bestimmte Laufwerksanzeigen kurz aufleuchten.

11.3.4 Während einer Netzwerksuche verschwindet bei der Suche nach bestimmten Pfaden kurzzeitig der Verzeichnispfad. Außerdem kommt es vor, dass nur der Schriftartenname angezeigt wird (und nicht der Verzeichnispfad), wenn eine Schriftart gefunden wurde. Sie können sich darauf verlassen, dass der Font Manager nach Abschluss der Suche wieder voll unter Ihrer Kontrolle ist.

11.3.5 Wenn Sie den Suchpfad im Bearbeiten-Feld angeben (Sie geben den Pfad also manuell ein, nicht mithilfe des Windows-Browsers), wird beim Suchvorgang selbst zwischen Groß- und Kleinschreibung unterschieden. Dies gilt für Betriebssysteme, die immer zwischen Groß- und Kleinschreibung unterscheiden (z.B. Windows Vista) und für Betriebssysteme, die normalerweise nicht zwischen Groß- und Kleinschreibung unterscheiden (z.B. Windows XP). Wenn Sie sich wegen der Groß- und Kleinschreibung nicht sicher sind, sollten Sie den Suchpfad mithilfe des Browsers angeben.

11.4 PROBLEME BEI DER INSTALLATION VON SCHRIFTARTEN
11.4.1 Derzeit unterstützt der Font Manager folgende Formate nicht:
- MultiMaster-Variante von Typ 1-Schriftarten
- TrueType Collection (xxx.TTC)
- OpenType (xxx.OTF)
- alle Bitmap-Schriftarten
- Vista CompositeFont (xxx.CompositeFont)

11.5 PROBLEME BEI DER DEINSTALLATION VON SCHRIFTARTEN
11.5.1 Der Font Manager verursacht ein Problem, wenn gleichzeitig die Funktion "Fontset/Autoaktivierung" des Adobe Type Manager (ATM) aktiviert ist. Wenn der Font Manager eine Schriftart vom Typ 1 entfernt, reinstalliert der ATM diese beim Neustart von Windows. Grund hierfür ist eine erweiterte Einstellung aus ATM 4.0, namens "Type 1 Font-Steuerungen/Autoaktivierung aktivieren." Diese Funktion ist standardmäßig aktiviert. Sie sollte jedoch deaktiviert sein.

Unter Windows Vista besteht dieses Problem nicht, da bei diesem Betriebssystem Typ 1-Schriftarten nicht mehr vom Adobe Type Manager verwaltet werden.

11.7 SCHRIFTARTENPROBLEME IN DER VORSCHAU ANZEIGEN
11.7.1 Fett oder kursiv formatierte Schriftartenzeichen werden in der Vorschau abgeschnitten. Bei Auswahl eines niedrigeren Schriftgradwertes verschwindet das Problem normalerweise.

11.7.2 Wenn eine installierte Schriftartenfamilie mit vier Schriftschnitten mit hoher Schriftgradeinstellung (z.B. 200 pt) in der Vorschau angezeigt dargestellt werden soll, wird das Vorschaufenster maximiert. Es können jedoch nicht alle Schriftschnitte angezeigt werden, weil sie über den unteren Bildschirmrand hinausragen. Verringern Sie die Schriftgradeinstellung, bis alle Schriftschnitte im Vorschaufenster vollständig sichtbar sind.

11.10 ALLGEMEINE PROBLEME BEI DER NUTZUNG DES FONT MANAGER
11.10.1 Der Font Manager darf nur dann verwendet werden, wenn der Schriftartenblock der Windows-Systemsteuerung geschlossen ist.

11.10.2 Wenn der Font Manager für die Ausführung einer bestimmten Operation sehr lange benötigt, können kurzzeitig die Listen "Für die Installation verfügbare Fonts" und "Unter Windows installierte Fonts" verschwinden. Die Fenster werden wieder angezeigt, wenn die Operation abgeschlossen ist.
