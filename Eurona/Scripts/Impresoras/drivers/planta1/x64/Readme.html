Getting Started

(1) Installing the printer driver

(1)-1 Windows 7/Vista/Server 2008 R2

Be sure to close all applications before upgrading the driver.
Note: This procedure requires administrator privileges (admin account or "Run as Administrator").

  • Step 1. From the "Start" menu, select "Devices and Printers" > "Add a Printer".
  • Step 2. The "Add a Printer" dialogue will be displayed asking "What type of printer do you want to install?" Select "Add a local printer".
  • Step 3. The "Choose a printer port" dialogue will be displayed. Create a new "Standard TCP/IP Port" and click "Next".
  • Step 4. Enter the "DEVICE" IP address and uncheck "Query the printer and automatically select the driver to use". Click "Next".
  • Step 5. The "Install the printer driver" dialogue appears. Click "Have Disk".
  • Step 6. Browse to the folder containing the driver files (typically the CD-ROM or an extracted archive downloaded from the website) and click "OK".
  • Step 7. Follow the rest of the Wizard's instructions to complete the installation.

(2) Upgrading the printer driver

Please follow these installation steps to update the printer driver from old version to new version.
Be sure to close all applications before upgrading the driver.
Note: This procedure requires administrator privileges.
Attension: If you want to upgrade the printer driver, use the same name printer driver.

  • Step 1. Open the "Printers" folder (or "Printers and Faxes", "View Devices and Printers") from the "Start" menu or "Control Panel".
  • Step 2. Open the Properties of the printer to be upgraded.
  • Step 3. Select "Advanced" tab > "New Driver" button. Click "Next >".
  • Step 4. Select "Have Disk..." then browse to the folder containing the extracted printer driver (you chose at step 2) INF file. Click "Open".
  • Step 5. Click "OK" and select the printer model. Click "Next >".
  • Step 6. Once the installation is complete, click "Finish".
  • Step 7. Click "OK" to close the printer properties dialog box.
  • Step 8. To finalize upgrade, log off and log on again.
After completing above operation, you can use printer driver of new version.

(3) Uninstalling the printer driver

(3)-1 Windows 7/Vista/Server 2008 R2

Be sure to close all applications before upgrading the driver.
Note: This procedure requires administrator privileges (admin account or "Run as Administrator").
  • Step 1. Control Panel > Printers
  • Step 2. Locate the logical printer corresponding to the driver that is to be uninstalled and delete it.
  • Step 3. Delete the driver from Server Properties > Driver tab.
  • Step 4. Log the current user account off.

(3)-2 Windows XP and 2K

Be sure to close all applications before upgrading the driver.
Note: This procedure requires administrator privileges.
  • Step 1. Open the "Printers" folder (or "Printers and Faxes") from the "Start" menu or "Control Panel".
  • Step 2. Delete the logical printer corresponding to the driver that is to be uninstalled.
  • Step 3. File > Server Properties > Driver tab.
  • Step 4. Select the driver to be uninstalled.
  • Step 5. Press the Delete key.
  • Step 6. Log the current user account off.

Other important points

[General Issue]

  • Banner pages cannot be printed using the source tray that is selected to print "Chapter Page". Please use another source tray to print banner pages.
  • If you are printing a letter size file in Microsoft Office Word 2010 and printed on A4 size.In that case, uncheck [Options] - [Advanced options] - [Print]-[Scale content for A4 or 8.5 x 11" paper sizes] from the menu in Word.
  • If odd-paged document is printed as the following setting at Adobe Reader X, an extra page might be added and the added page might be incremented in the printer's counter.
        Application settings:
    Copies : 2 or more
    Collate : Not Off
        Driver settings:
    Duplex : Not Off
    If application's copies is 1 and driver's copies is 2 or more, extra page is not added.
  • In the environment to which the computer name is set by 16 bytes or more, the host name with banner page print is printed only up to 15 bytes.
  • In the case of specific application, only one part will be printed even if the combination of post-processing features and number of copies by from the driver set of multiple. Post-processing capability refers to the following items.
    • Staple
    • Punch
    • Fold
    • Sort
    • Perfect Binding etc.
      The example of specific application : Microsoft Office Project, Microsoft Office Visio

[OS/Windows Related Issues]

  • The driver's "Web Image Monitor" button might not work in an IPv6 environment.

[Driver Setting Related Issues]

  • The use of Reverse Order is not supported with the Duplex, Front Cover, Back Cover, Chapter Pages, Slip Sheet functions.
  • In the case of specific application, may not print number of copies that set by the driver.
      [Avoidance]Set number of copies by applications.
        The example of specific application :
    • Adobe Reader X
    • Adobe Acrobat X
    • Adobe Illustrator CS5
    • Adobe InDesign CS5
    • Adobe InDesign CS5.5
  • If the Staple setting is "2 at Center" or if the "Reverse Order"setting is On, the driver will add an extra page to the end of odd-paged duplex jobs. The added page will be incremented in the printer's counter. There is no way to prevent this.

[Printing Issues]

  • Additional Page Count in Duplex Printing. An extra page is added to the end of odd-paged duplex jobs. The added page might be incremented in the printer's counter.
  • Thin lines printed at high resultion (1200dpix1200dpi) might be difficult to see in the printed output. In such cases, use of a lower resolution is recommended.
  • Inconsistencies during the cross pattern in raster mode to print specific data.
    In this case, set the vector mode.
  • Depending on the combination of text and font settings for watermark, text might not be printed.
    In such case, a change of font is recommended.

[B/W print with CMYK setting Issue]

  • If the "Color/ Black and White" setting is "Color" some purely BW objects might be printed using CMYK toners (composite black). If the "Color/ Black and White" setting = "Black and White", all objects will be printed using K toner (pure black) only.
